## §4. AI 行业动态影响课程方向

- **Claude Sonnet 5 发布（[Anthropic, 2026-06-30](https://www.anthropic.com/news/claude-sonnet-5)）**：最具 Agent 能力的 Sonnet；Opus 4.8 仍是旗舰（agentic coding 69.2%），但 Sonnet 5 以更低成本提供接近性能，Claude Code 默认切换。**对 JR 影响**：Bootcamp 所有 Claude API 演示应切换至 Sonnet 5（成本直接降 40%）；"模型选型"课程章节需更新价格/性能对比，把 Sonnet 5 作为"日常 workhorse"、Fable 5 作为"高难任务备选"讲清楚。
- **Fable 5 出口管制解除（[The Rundown AI, 2026-07-01](https://www.therundown.ai/p/sonnet-5-ships-as-washington-frees-fable)）**：美国商务部持续 18 天对 Fable 5（Mythos 级）的出口限制于 7月1日解除。**对 JR 影响**：Fable 5 此前限制了部分非美地区 API 访问，现可对澳洲学员完整演示；AI 政策与出口管制是新的合规话题，JR AI Engineer Bootcamp 可加一节"AI 地缘政治与合规"（10~15 分钟），实际案例就在这里。
- **PwC 2026 Global AI Jobs Barometer（[PwC, 2026-06](https://www.pwc.com/gx/en/news-room/press-releases/2026/pwc-2026-ai-jobs-barometer.html)）**：AI 专项岗位增速 = 总体就业市场的 8 倍；75%+ 招聘明确要行业专精（金融/医疗/法律+AI）。**对 JR 影响**：泛 AI 课程压力增加，JR 的"AI + 行业"垂直方向（如金融 AI、医疗 AI）应加速布局；求职辅导也应强化"行业 + AI 工具"组合叙事，不能只讲通用技能。
- **Coursera 进驻 Microsoft 365 Copilot（[Coursera Blog, 2026-06](https://blog.coursera.org/coursera-launches-the-first-learning-agent-in-microsoft-365-copilot-enabled-by-apps-sdk/)）**：学习内容嵌入工作流，是"在工作中学习"的具体产品化。**对 JR 影响**：竞品已在探索工作流嵌入学习，JR 若有企业培训方向，可关注类似集成（未来 Notion/Slack 里的微学习单元）；短期内无需跟进，但值得注意趋势。

## §6. 对 JR Academy 的建议

1. **本周将 Bootcamp 所有 Claude 演示切换至 Sonnet 5**：$2/$10/Mtok 促销至 8月31日，是用旗舰级性能模型做演示成本最低的窗口。同步更新课程"模型选型"章节：Sonnet 5 = 日常 workhorse（低成本），Fable 5 = 高难 Agent 任务（按需调用）；两者对比实操是本月最好的教学切入点。

2. **加一节"AI 政策与地缘合规"（15 分钟，AI Engineer Bootcamp）**：Fable 5 出口管制 18 天解除是活教材，学员今后做 AI 产品落地必然碰到合规墙（GDPR、出口管制、模型许可证）。这个话题 DeepLearning.AI / Coursera 都没教，JR 先做是差异化点；案例即是本周事件，无需额外调研。

3. **求职模块叙事改成"AI + 行业专精"**：PwC 数据显示 75%+ 职位要行业专精，泛 AI 技能被筛掉的风险上升。JR 的简历辅导和面试准备应帮学员建立"我是 [金融/电商/医疗] + AI Engineer"的定位，而不是"我学过 LangChain/RAG"——后者已是基本门槛，不再是亮点。
