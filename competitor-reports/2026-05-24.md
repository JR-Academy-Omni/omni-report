## 4. AI 行业动态影响课程方向

本周三大 newsletter（The Rundown AI / TLDR AI / Ben's Bites）共同主线：Google I/O 2026 全平台 Agent 化 + Anthropic 并购拓宽开发者工具生态。

- **Google I/O 2026（2026-05-19~20，直播 + 按需会话从 5 月 21 日开放）** — Google 宣布 Gemini 3.5 系列，将开发平台 Antigravity 升级至 2.0：Antigravity CLI 可一键启动专用子 agent，内置跨平台终端沙箱、凭证屏蔽、强化 Git 策略；同期发布 [WebMCP](https://developers.googleblog.com/all-the-news-from-the-google-io-2026-developer-keynote/)：浏览器原生工具暴露标准（Chrome 149 开始 origin trial），对 AI 工程师课程影响：①"工具链选型"章节需新增 Gemini 3.5 vs Claude vs GPT 对比实验；② Antigravity CLI 的 agent 编排方式与 LangGraph/LangSmith 是直接竞争关系，Agent 章节应做工具对比。[Google I/O 2026 全文](https://io.google/2026/)
- **Anthropic 收购 Stainless（2026-05-18）** — Stainless 负责生成所有官方 Anthropic SDK 及 MCP server tooling；收购后 MCP 生态工具将加速迭代。**对 JR 影响**：AI Engineer Bootcamp 的 MCP 章节需持续跟进工具变化，建议本季度在课程仓库里专设"MCP 工具更新日志"供讲师参考。[收购公告](https://www.anthropic.com/news/anthropic-acquires-stainless)
- **Anthropic 首超 OpenAI ARR（2026年5月初）** — Anthropic ARR 达 $300 亿美元，OpenAI $240 亿，首次超越。**对 JR 影响**：课程叙事中"Claude vs GPT"工具选型讨论可用此数据支撑"Claude 正在成为企业首选"的论点，增强 Claude Code 章节可信度。

## 6. 对 JR Academy 的建议

1. **本周内在 AI Engineer Bootcamp 新增"多模态 Agent 评估"Lab**：DeepLearning.AI + Google 已出免费课程（May 20），教 SigLIP 评估 AI 生成图像/视频；JR 无此模块形成竞争空白。建议设计 1 个 Lab：学员用 SigLIP 对比自己 prompt 生成的图像与预期文字描述的相似度，输出评估报告——中文授课 + 对应澳洲 AI Engineer 岗位 JD 说明做差异化。
2. **招生页本周更新薪资数据 + 对标 Coder Academy 定价**：SEEK AU AI Engineer 在招 2,448 个（+11%），悉尼高级岗薪资 AUD $230k+；同时 Coder Academy 降至 $15,000，与 GA 同价位——JR 招生页应明确列出自身区别点（如中文支持、澳洲本地就业辅导、课程深度），避免被价格直接对比。
3. **在课程仓库新建"MCP 工具更新日志"，每两周维护一次**：Anthropic 收购 Stainless 后 MCP server tooling 迭代加快；Google I/O 同期发布 WebMCP（浏览器原生工具标准）——两条主线叠加，Agent 工具生态本季度将高频变动。建议在 GitHub 仓库课程目录下新建 `TOOL_CHANGELOG.md`，讲师每两周更新一次 MCP/工具链变动，防止课程内容落后市场超过 4 周。
