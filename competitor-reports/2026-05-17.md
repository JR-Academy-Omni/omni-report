## 4. AI 行业动态影响课程方向

本周三家 newsletter（The Rundown AI / TLDR AI / Ben's Bites）共同主线：Anthropic 企业生态扩张 + OpenAI 开发工具链强化 + 模型架构创新。

- **Anthropic Claude Code 新增 agent view（2026-05-12 更新）** — 单 CLI 管理多 Agent session，smarter 模型选择（Opus/Sonnet 自动分配任务）；同周 Anthropic 宣布 6 月 15 日起程序化调用独立计费（[InforWorld](https://www.infoworld.com/article/4171274/anthropic-puts-claude-agents-on-a-meter-across-its-subscriptions.html)）。**对 JR 影响**：Vibe Coding Bootcamp 需同步更新 Claude Code 实操章节，加入 agent view 多 session 操作；同时向学员说明 6 月 15 日后 Agent SDK 计费变化，避免课后用量账单超预期。
- **OpenAI "Work with Codex from anywhere"（2026-05-15）** — Codex CLI 可在任意终端环境调用 OpenAI 模型执行编程任务；配合 5 月 7 日发布的实时语音 API（reasoning/翻译/转录），OpenAI 的 developer 工具链本周连续出牌。**对 JR 影响**：AI Engineer Bootcamp 的"工具链选型"章节应补充 Codex CLI vs Claude Code 对比，让学员实际用过两者后做选择。
- **SubQ 1M-Preview（Subquadratic，2026-05-05）** — 首个商用 subquadratic sparse attention 架构 LLM，原生 12M token 上下文窗口，成本为同级 frontier 模型的 1/5，大规模 attention 速度提升最高 52× — [WhatLLM.org 报道](https://whatllm.org/blog/new-ai-models-may-2026)。**对 JR 影响**：RAG + 长文档处理章节可增加"无需向量数据库的超长上下文方案"实战对比（超长上下文 vs RAG 的成本/精度 tradeoff）。
- **Anthropic × PwC 扩大战略联盟（2026-05-12）** — PwC 将用 Claude 为企业客户构建技术、执行交易、重构企业职能 — [Anthropic](https://www.anthropic.com/news/pwc-expanded-partnership)。**对 JR 影响**：AI Engineer Bootcamp 的"企业 AI 落地"模块可引用 PwC + Claude 合作为真实案例，说明大型咨询公司如何批量部署 AI 工程师技能。
