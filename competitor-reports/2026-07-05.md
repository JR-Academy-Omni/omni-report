## §4. 行业动态

- **Claude Sonnet 5 正式发布（[Anthropic, 2026-06-30](https://www.anthropic.com/news/claude-sonnet-5)）**：最强 Sonnet 级别，能力接近 Opus 4.8；新 tokenizer 同等输入产生 ~30% 更多 tokens；促销价 $2/$10 per Mtok 至 2026-08-31。**对 JR 影响**：Claude Code 默认已切换 Sonnet 5，JR Vibe Coding Bootcamp 中 API 成本估算需更新；新 tokenizer 影响 context window 使用效率，应在课程中告知学员

- **GPT-5.6 三模型家族发布（OpenAI, 2026-06-26）**：Sol（旗舰）/ Terra（GPT-5.5 性能，半价）/ Luna（最快最便宜）。**对 JR 影响**：AI Engineer Bootcamp 的"模型选型"章节需重新梳理价格-性能矩阵；Terra 的"半价达到上一代旗舰性能"是企业选型的典型论据，适合出一节实战 lab

- **Claude Fable 5 管制风波全球解封（[TechCrunch, 2026-07-01](https://techcrunch.com/2026/06/30/anthropic-launches-claude-sonnet-5-as-a-cheaper-way-to-run-agents/)）**：美国出口管制导致 Fable 5 下线 18 天，7月1日恢复全球访问。**对 JR 影响**：AI 合规与出口管制是 2026 年企业 AI 部署的真实风险；建议在 AI Adoption 课程中增加"地缘政治与 API 合规"模块（已在上期建议，本期事件再次印证）

- **Claude Science Beta 上线（[Anthropic, 2026-07-04](https://www.anthropic.com/news/claude-science-ai-workbench)）**：面向科研的多 Agent 工作台，60+ 技能，覆盖基因组学/蛋白质组学/化学信息学。**对 JR 影响**：AI 进入专业科研场景，JR 可考虑面向澳洲高校研究员的短期工作坊（"如何用 Claude Science 加速文献综述"）

- **WAIC 2026（2026-07-17~20，上海）**：5 大生态版块，含 WAIC UP! 知识共享，[worldaic.com.cn](https://www.worldaic.com.cn/)；量子位、机器之心为官方媒体。**对 JR 影响**：大会前后中文 AI 学习内容热度将上升，适合 JR 在国内微信/B站提前布局相关 Bootcamp 招募内容

1. **本周内更新 API 成本估算 + 模型选型章节**：Claude Sonnet 5 和 GPT-5.6 Terra 同期发布，两款模型都在"更低价格达到上一代旗舰性能"——AI Engineer Bootcamp 的"选型与成本"lab 需要新的对比数据表；Claude Code 的 Sonnet 5 促销期到 8月31日，现在开课的学员可以在最低价格下完成整个 Bootcamp
