## 4. AI 行业动态影响课程方向

- **DeepSeek V4 Flash + Pro 发布（2026-04-24）** — 预览版上线，主打编码 benchmark 和 agentic 推理，Huawei Ascend 确认兼容 — [CNN](https://www.cnn.com/2026/04/24/tech/chinas-ai-deepseek-v4-intl-hnk) / [CNBC](https://www.cnbc.com/2026/04/24/deepseek-v4-llm-preview-open-source-ai-competition-china.html)。**对 JR 影响**：AI Engineer Bootcamp 的开源模型选型章节应新增 DeepSeek V4 Pro，与 Claude Opus 4.7 / GPT-5.5 横向对比。
- **Cursor 3 发布（2026-04 月内）** — 从"智能补全"升级为 agent-first 界面，开发者可直接将编码任务分配给 AI agent，不再逐行写代码 — [相关报道](https://kersai.com/ai-breakthroughs-april-2026-models-funding-shifts/)。**对 JR 影响**：Vibe Coding Bootcamp 的 Cursor 章节需完整重写，Demo 从"行内补全"切换到"task-level agent 交互"。
- **Databricks Unity AI Gateway（2026-04-15）** — 将 Unity Catalog 权限治理扩展到 MCP Server 层，AI agent 访问 LLM 的每一次调用均可审计 — [Databricks 博客](https://www.crescendo.ai/news/agentic-ai-news-and-developments)。**对 JR 影响**：AI Engineer Bootcamp 的 MLOps 章节应补充"Agent 治理 / MCP 权限控制"内容，这是企业级部署的关键卡点。
- **The Rundown AI / TLDR AI / Ben's Bites** — 本周三家 Newsletter 共同主题：Agentic workflow 取代单点 AI 功能（"every significant launch is a workflow design, not a feature add-on"）；DeepSeek V4 和 Cursor 3 均为重点报道。具体期刊链接：[The Rundown AI](https://www.therundown.ai/) / [Ben's Bites](https://news.bensbites.com/)
