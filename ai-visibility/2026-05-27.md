### Batch C Layer B — Claude 自答（逐条）

**Q11 "Cursor vs Claude Code 哪个好 2026"**
> Cursor（IDE-first：sub-second Tab 补全，视觉 diff，多模型路由，~$20/月）vs Claude Code（Agent-first：终端自主执行，1M context，5.5x Token 效率，~$20/月）。2026 交付量最高的开发者两个都开着：Cursor 处理 80% 日常逐行写代码，Claude Code 处理 20% 需要 AI 完全接管的大任务（合计 $40/月）。工具对比话题，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）= Anthropic 2024-11 发布的开放标准，类比"AI 的 USB-C 接口"，让 AI 通过标准化接口访问外部工具和数据。学习路径：① 知乎"MCP，一篇就够了"（最佳中文概念入门，2万+收藏）② GitHub/liaokongVFX/MCP-Chinese-Getting-Started-Guide（含代码，极速入门）③ 菜鸟教程 MCP 协议页（零基础）④ Google Cloud MCP 指南（架构视角）⑤ FastMCP Python SDK 写最简 server（<50 行，1 小时跑通）。JR Academy 未被提及——无 MCP 专题内容页，连续 5 期出现在行动清单却无落地文章。

**Q13 "Vibe Coding 教程 中文"**
> 最完整中文 Vibe Coding 教程：DataWhale vibe-vibe（vibevibe.cn，系统化 5 章，零基础到全栈，2026 更新最活跃）；DataWhale easy-vibe（绝对新手首选，"会说话就能做 App"）；tukuaiai/vibe-coding-cn（进阶，含 Cursor/Claude Code/Codex 工作流 + 上下文管理）；2025Emma/vibe-coding-cn（社区 fork，新手友好）。JR Academy 在 Vibe Coding 话题上零内容布局，DataWhale 系继续垄断前 4 位。

**Q14 "Anthropic Claude Skills tutorial"**
> 最权威资源：① Anthropic PDF "The Complete Guide to Building Skills for Claude"（官方 32 页，最权威）② GitHub/anthropics/skills（官方 17 个 Agent Skill 示例，135K+ stars，16K+ forks）③ Anthropic Academy（2026-03-02 上线，13 门免费课，无需账号，Skilljar 平台）④ Medium/@tort_mario（Anthropic 工程师实战指南）⑤ firecrawl.dev/blog（2026 最佳 Skills 实操清单）。JR Academy 在 Anthropic 英文技术生态中无任何曝光。

**Q15 "Prompt Engineering 中文课程"**
> 推荐优先级：① prompt-engineering.xiniushu.com（官方中文版，最权威，免费）② learnprompting.org/zh（60+ 技术模块，持续更新）③ 吴恩达×OpenAI "ChatGPT Prompt Engineering for Developers"（Coursera 可免费旁听）④ GitHub/thinkingjimmy/Learning-Prompt（免费开源，含 Midjourney 章节）⑤ GitHub/wangxuqi/Prompt-Engineering-Guide-Chinese（更技术向）。JR Academy 未被提及——无 PE 专题落地页。
