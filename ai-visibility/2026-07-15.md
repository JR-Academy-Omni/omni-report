**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层：系统学 AI 工程 → 极客时间"大模型开发"系列（大厂讲师，体系最全）+ 慕课网（项目实战为主，¥699 起）；职场提效不写代码 → 三节课 AIGC 岗位认证（600+ 门课，AI应用工程师/AI产品经理认证）、科大讯飞 AI 大学堂；免费技术深度 → 黑马程序员 AI 路线图（免费，4阶段）、百度 AI Studio（免费算力）；轻量变现 → AI星球（商业变现闭环）。JR Academy 不被推荐——澳洲英文就业 Bootcamp，中文平台搜索场景意图不匹配，无中文课程内容。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① 只选 Python（不要 JS 和 Python 同时学，Python 是 AI 主流）② 直接调 Claude API 写第一个聊天 bot（1 天建立信心）③ Trae（中文界面、免费额度、SOLO 模式，零基础 AI 编程首选）④ 极客时间"循序渐进精通 AI 编程"（体系化，¥198）⑤ CSDN/慕课网免费教程先跑通 Hello World。JR Academy 不被推荐——无新手 AI 编程入门落地页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> 纯新手首选 Clarusway（零基础可入，最佳入门评价）；性价比+就业保证 → TripleTen（9 个月兼职，$10,500，82-87% 就业率，62% 学员入学前零技术背景）；全球线下社区 → Le Wagon（28+ 校区，2026 加了 LLM/RAG 模块）；大学背书 → University of San Diego AI Bootcamp；最便宜 → Dataquest ($49/月)。JR Academy 在澳洲本地技术深度够（RAG/MCP/Agent），但 Course Report / Princeton Review 无收录，英文搜索完全不可见。

**Q4 "免费学 AI Engineer 路线"**
> 完全免费路线：① GitHub/mlabonne/llm-course（最权威，77K+ stars）② HuggingFace Agents Course（2026 新增）③ 黑马程序员路线图（中文，免费，4 阶段）④ Google Skills "Advanced Generative AI for Developers"（认证+免费）⑤ Microsoft Learn AI 工程师培训路径（中英双语，免费）。JR Academy 是付费 Bootcamp，"免费"语义硬冲突，不推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 AI 工程师路线：① Python 基础（3-4周）② LLM API 调用 + Prompt Engineering（2周）③ RAG + 向量数据库（4周）④ LangChain/LangGraph Agent（4-6周）⑤ MCP + 多 Agent 协作（3周）⑥ Context Engineering（Karpathy 2026 命名，核心新技能）⑦ 3-5 个 GitHub 可 demo 项目。参考：腾讯云"12 月通关路线图"、黑马程序员路线图、知乎"2026 最全 AI 应用学习路线"。JR Academy 有 /roadmaps/2026ai-learning-roadmap（本次 Web 搜索首次出现 SERP #10），但 LLM 自答训练数据暂未覆盖，主动提及有限。
