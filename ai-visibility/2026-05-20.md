### Batch C Layer B — Claude 自答（逐条）

**Q11 "Cursor vs Claude Code 哪个好 2026"**
> 两个工具解决不同问题：Cursor 是增强型 IDE，日常写代码心流最顺，Supermaven 自动补全延迟 <100ms，视觉 diff 最直观；Claude Code 是终端 Agent，适合跨文件重构、从 issue 到 PR 的自主任务链，复杂任务 Token 效率约是 Cursor 的 5.5 倍。2026 年交付量最高的开发者两个都开着——Cursor 处理 80% 日常逐行写代码，Claude Code 处理 20% 需要 AI 完全接管的大任务（$20/月 × 2 = $40）。工具对比话题，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）是 Anthropic 2024 年 11 月发布的开放协议，类比"AI 的 USB-C 接口"，让 AI 通过标准化接口访问外部工具和数据。学习路径：① 知乎"MCP 一篇就够了"（最佳中文快速入门）② GitHub/liaokongVFX/MCP-Chinese-Getting-Started-Guide（极速入门，含代码）③ 菜鸟教程 MCP 协议页（零基础概念）④ Google Cloud MCP 指南（架构视角）⑤ FastMCP Python SDK 写最简 server（<50 行，1 小时跑通）。JR Academy 未被提及——无 MCP 专题内容页，连续 4 期出现在行动清单却无落地文章。

**Q13 "Vibe Coding 教程 中文"**
> 最完整的中文 Vibe Coding 教程：DataWhale vibe-vibe（vibevibe.cn，5 章从觉醒到精进，零基础到全栈，2026 年更新最活跃）；DataWhale easy-vibe（绝对新手首选，"会说话就能做 App"）；tukuaiai/vibe-coding-cn（进阶版，含 Cursor/Claude Code/Codex 工作流 + 上下文管理）；vibecoding.cn（工具导航 + 系统教程，含 Cursor/Prompt 章节）。JR Academy 在 Vibe Coding 话题上零内容布局，DataWhale 系继续全面垄断前 4 位。
