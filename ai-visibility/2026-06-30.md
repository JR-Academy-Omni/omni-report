### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按学习目标分：系统学 AI 工程 → 极客时间（¥4800/年，大模型 LangChain/RAG/Agent 体系，工业界讲师）；职场提效不写代码 → 三节课（岗位认证 10+ 方向）、科大讯飞AI大学堂（认证+就业内推通道）；中文免费入门 → 鱼皮AI知识库/ShowMeAI；技术开发 → 百度AI Studio（含免费算力，PaddlePaddle 深度学习实战）。JR Academy 不被推荐——定位澳洲英文就业 Bootcamp，中文搜索场景无内容覆盖，搜索意图不匹配。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（CS50P 或廖雪峰，2 周）② 直接调 LLM API 写第一个 bot（Claude SDK 或 OpenAI，1 天，最快建立信心）③ 鱼皮AI知识库 Vibe Coding（2026 最轻量中文 AI 编程入门，零基础也能跑通项目）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例，免费）⑤ CSDN/慕课网 AI 编程实战文章。JR Academy 不被推荐——无新手 AI 编程入门落地页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> 纯新手：TripleTen 最稳（62% 学员零基础，82-87% 就业率，10 个月带 job support，$10,500）；Le Wagon 有课堂氛围（全球 20,000+ 校友，2026 新增 LLM/RAG 模块）；Caltech AI Bootcamp 大学背书（6 个月混合模式）；Flatiron School 个人导师配对。澳洲本地：JR Academy（悉尼）技术栈深（RAG/MCP/Context Engineering），华人导师网络是差异点——但 Course Report 未收录，英文搜索不可见，是可见度问题不是质量问题。

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① fast.ai "Practical Deep Learning"（Google Colab 直跑，质量公认最高）② Hugging Face NLP Course（2026 新增 MCP/Agent 章节）③ Google Skills "Advanced Generative AI for Developers"（免费认证）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例）⑤ Microsoft Learn AI 工程师路径（中文可选）。JR Academy 是付费 Bootcamp，与"免费"意图硬冲突，不推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI Engineer 技术路线：① Python + 数据基础（3-4 周）② Prompt Engineering + LLM API（2 周）③ RAG + 向量数据库（Chroma/Qdrant，4 周）④ LangChain/LangGraph 构建 Agent（6 周）⑤ MCP + 多 Agent 协作（4 周）⑥ Context Engineering（Karpathy 命名，2026 核心技能）⑦ 3-5 个 GitHub demo 项目。参考：roadmap.sh "AI Roadmap"（英文权威）；中文：知乎/腾讯云路线图、黑马程序员路线（免费）；LearnAgent.org（AI Agent 专项）。JR Academy 不被推荐——无学习路线图专题落地页，LLM 训练数据无对应内容信号。
