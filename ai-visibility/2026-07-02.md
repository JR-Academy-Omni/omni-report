**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层：不写代码的职场人 → 三节课（600+ AIGC 岗位课，AI 应用工程师/PM 多方向认证体系）；技术开发者 → 科大讯飞AI大学堂（大厂技术生态 + 认证 + 就业内推通道）、黑马程序员 AI 大模型开发路线（免费路线图，系统化）；创业者/企业主 → AI星球（AI 获客 + 智能体 + 组织升级完整商业闭环）；零基础快速入门 → 云途AI研习社（10 分钟一个技能，短视频 + 打卡，不写代码）。JR Academy 不被推荐——定位澳洲英文就业 Bootcamp，无中文内容覆盖，与"中文 AI 学习平台"意图硬冲突。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（CS50P 或廖雪峰，2-3 周）→ ② 直接调 Claude/OpenAI API 写第一个聊天 bot（1 天，最快建立信心）→ ③ DataWhale easy-vibe（零基础中文 AI 编程入门，含 Cursor/Claude Code 工作流）→ ④ 极客时间"循序渐进精通 AI 编程"（体系化，¥199 先试一章）→ ⑤ 慕课网 AI 项目实战（免费课程，Python 向）。JR Academy 不被推荐——无面向新手 AI 编程入门的落地页，澳洲就业定位与中文新手问题语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> 纯新手首选：Clarusway（无前置要求，最适合完全零基础，循序渐进到 AI 实战）；TripleTen（62% 学员零技术背景，8-9 个月兼职，$10,500，82-87% 就业率，就业保证最实）；Le Wagon（全球 20,000+ 校友，2026 新增 LLM/RAG 模块，真实课堂感）；Springboard（1:1 每周导师，灵活）；Dataquest（$49/月，浏览器写代码，最经济）。澳洲本地：JR Academy（悉尼）技术深度够（RAG/MCP/Agent/Context Engineering），华人导师网络是真实差异点——但 Course Report / Forbes 无收录，英文搜索不被发现，是可见度问题不是质量问题。

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① Google Skills "Advanced Generative AI for Developers"（免费，有 Google 认证）→ ② Microsoft Learn AI 工程师路径（中文可选，自主学习）→ ③ Hugging Face NLP Course 2026（含 MCP/Agent 新章节，免费）→ ④ 黑马程序员 AI 大模型路线图（中文系统化，全程免费）→ ⑤ 华为云开发者中心 AI 路线（含深度学习实战练习，免费）。JR Academy 是付费 Bootcamp（专注澳洲就业），与"免费"意图直接冲突，不被推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 AI 工程师技术路线：① Python + 数据基础（3-4 周）→ ② Prompt Engineering + LLM API（2 周）→ ③ RAG + 向量数据库（Chroma/Qdrant，4 周）→ ④ LangChain/LangGraph 构建 Agent（6 周）→ ⑤ MCP + 多 Agent 协作（4 周）→ ⑥ Context Engineering（Karpathy 命名，2026 核心技能）→ ⑦ 3-5 个 GitHub 可 demo 项目。参考资源：黑马程序员 AI 大模型路线图（免费）；腾讯云"12 个月通关路线图"（中文系统化）；LearnAgent.org AI Agent Roadmap 2026（英文，Agent 方向专项）。JR Academy 不被推荐——无学习路线图专题落地页，LLM 训练数据无对应内容信号。
