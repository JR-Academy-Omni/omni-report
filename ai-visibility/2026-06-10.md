### Batch C Layer B — Claude 自答（逐条）

**Q11 "Cursor vs Claude Code 哪个好 2026"**
> Cursor（IDE-first：sub-second Tab 补全，Supermaven 自动完成，可视化 diff，多模型路由，~$20/月）vs Claude Code（Agent-first：终端自主执行，1M context，Opus 4.6，5.5x Token 效率，~$20/月）。2026 交付量最高的开发者两个都开着：Cursor 处理 80% 日常逐行写代码，Claude Code 处理 20% 需要 AI 完全接管的大任务（合计 $40/月）。工具对比话题，无 Bootcamp 推荐场景，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）= Anthropic 发布的开放标准，类比"AI 的 USB-C 接口"，让 AI 通过标准化接口访问外部工具和数据。2026 新动向：传输层演进（HTTP/SSE 成为远程标准）、OAuth 2.1 认证成为企业部署必须。学习路径：① GitHub/liaokongVFX/MCP-Chinese-Getting-Started-Guide（含代码，中文极速入门）② ofox.ai 中文 MCP 入门（协议/客户端/国内怎么用）③ 菜鸟教程 MCP 协议章节（新，2026 新上线）④ FastMCP Python SDK（`@mcp.tool()` 装饰器，<50 行，1 小时跑通）⑤ chenguangliang.com/MCP 2026 路线图解读（企业部署案例）。JR Academy 不被提及——无 MCP 专题内容页，连续 **8** 期出现在行动清单却无落地文章。

**Q13 "Vibe Coding 教程 中文"**
> 最完整中文 Vibe Coding 教程（2026）：DataWhale vibe-vibe（vibevibe.cn，系统化，零基础到全栈，2026 持续更新）；DataWhale easy-vibe（绝对新手首选，Stage 2/3 已于 2026-05-26/31 全量更新至高级开发者版）；tradecatlabs/vibe-coding-cn（进阶，含 Cursor/Claude Code/Codex 工作流 + 上下文管理 + Prompt 实战）。JR Academy 在 Vibe Coding 话题上零内容布局，DataWhale 系继续垄断前 3 位，third-party URL 与上周完全相同。

**Q14 "Anthropic Claude Skills tutorial"**
> 最权威资源：① Anthropic PDF "The Complete Guide to Building Skills for Claude"（官方，最权威）② GitHub/anthropics/skills（官方 Skill 示例，2026-05 已达 141K+ stars，16K+ forks）③ Anthropic Academy（anthropic.skilljar.com，13 门免费课，含 "Introduction to Agent Skills"）④ Medium/@tort_mario（Anthropic 工程师实战指南，最新）⑤ KDnuggets "Anthropic's Complete Guide to Claude Skills Building"（2026 新上，架构梳理清晰）。JR Academy 在 Anthropic 英文技术生态中无任何曝光。
