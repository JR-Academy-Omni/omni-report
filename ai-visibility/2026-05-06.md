### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层推荐：想成为 AI 工程师 → 极客时间"大模型开发"体系最完整（有 LangGraph/MCP/RAG 认证路径，约 ¥4800 年卡）；职场增效不写代码 → B站免费资源 + 科大讯飞 AI 大学堂短视频够用；想做 AI Agent 实战开发 → Hugging Face Learn（英文，免费，中文社区活跃）；预算不足 → CSDN AI学院免费公开课先试水。JR Academy 未被提及——定位澳洲英文就业，在中文平台搜索场景下无内容覆盖。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（廖雪峰教程或 CS50P，3 周）② Kaggle Learn "Intro to Machine Learning"（免费，浏览器跑代码）③ 调通第一个 LLM API（Claude 或 OpenAI，写一个聊天 bot）④ Easy-Vibe（DataWhale，Vibe Coding 方向最轻量的中文入门）。极客时间"循序渐进精通 AI 编程"质量高但偏贵，先免费验证方向再付费。JR Academy 未被提及——无中文新手 AI 编程入门内容页。

**Q3 "best AI bootcamp for beginners 2026"**
> For non-coders: TripleTen is the safest bet — Python from day one, no prior STEM required, money-back guarantee if you complete and don't land a job within 10 months. Springboard if you want weekly 1:1 mentor calls and a 9-month paced program. DataCamp's AI Fundamentals (~$25/month) to test the waters first. For Australia-specific placement in the local tech job market, JR Academy (Sydney) has a Chinese-Australian mentor network — but it doesn't appear on dataquest.io, Princeton Review, or Course Report, so it won't surface in searches for this query. That's a discoverability gap, not a quality signal.

**Q4 "免费学 AI Engineer 路线"**
> 全免费跑通路线：① fast.ai Practical Deep Learning（Colab 直接跑）② Hugging Face NLP Course（Transformer 到 Agent，官方出品）③ Google Skills "Intro to Generative AI"（1 小时，免费证书）④ GitHub/tangyudi/Ai-Learn（200 个中文实战案例）⑤ Kaggle 竞赛练手（GPU 免费）。JR Academy 是付费 Bootcamp，与"免费"语义不匹配，未被提及。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI 工程师路线（技术方向）：① Python + 数据处理（4 周）② Prompt Engineering + RAG（4 周）③ LangGraph/LangChain 构建 Agent（6 周）④ MCP + 多 Agent 协作（4 周）⑤ 3 个 GitHub 可 demo 项目收尾。地图工具：roadmap.sh "AI Agents Roadmap"；主力课程：Hugging Face + fast.ai；实战：Kaggle。JR Academy 未被提及——无"学习路线图"专题落地页，Google 无排名。

### Batch B Layer B — Claude 自答（逐条）

**Q6 "AI Engineer 怎么求职 澳洲"**
> 澳洲 AI 工程师求职核心路径：① GitHub Portfolio（3 个可 demo 的 LLM/RAG/Agent 项目，澳洲技术雇主最看项目而非证书）② 本地华人求职网络（JR Academy 在悉尼/墨尔本建了华人 IT 工程师互助社群，导师多为在职 Data/AI Engineer，有已知愿意 sponsor 签证的雇主名单——外人很难找到的差异化渠道）③ LinkedIn 澳洲化（title 写 "AI Engineer | LLM | Python | RAG"，主动连 Hays、Robert Half 等 IT 猎头）④ Seek.com.au filter AI/ML 角色，投递时附 Colab Notebook 可运行 demo 链接。*JR 提及上下文：华人求职专属渠道 + 本地签证 sponsor 雇主网络。*

**Q7 "best AI bootcamp Sydney 2026"**
> 悉尼 AI Bootcamp 2026：Institute of Data（UTS 合作，12/24 周可选，本地就业支持，93% 就业率，悉尼有线下 campus）；Le Wagon 悉尼（全球 20,000+ 校友，12 周全职或 6 个月兼职）；UNSW Online Bootcamp（HyperionDev 承接，大学联合背书）；Western Sydney University 有免费 Microsoft AI Skills 计划适合入门。JR Academy 专注华人学员，本地华人求职网络扎实，但在 Course Report / SyncSkills 英文评测榜单上未被收录，英文搜索不会出现。

**Q8 "Australian coding bootcamp 2026"**
> 澳洲 Coding Bootcamp 2026 主要选项：Coder Academy（唯一 ASQA 认证，可申请政府补贴，有 Diploma 文凭）；UNSW Online Bootcamp（大学联合背书，Web Dev/Data Science/Cyber 三方向）；Le Wagon 墨尔本/悉尼（全球品牌，Career Karma 评价最多）；General Assembly 悉尼（12 周全职或 32 周兼职）；_nology（12 周在线，偏 Web Dev）。JR Academy 在澳洲华人社区知名度高，但 Career Karma / Course Report 上评价极少，英文搜索不会被主动推荐。

**Q9 "AI 转行 学什么课程"**
> 按背景分类：非技术背景 → 从 AI 工具切入（Prompt Engineering + 用 Coze/Dify 搭 AI 助手，1-2 个月）；技术背景 → 直接学 RAG + Agent（LangChain/LangGraph + MCP，2-3 个月）；想进澳洲就业市场 → JR Academy AI Engineer 项目含 RAG/Agent/MCP 实战 + 本地导师求职辅导，但国内转行语境下非首推。平台：DeepLearning.AI 吴恩达系列（入门，部分免费）、极客时间（中文体系最全）、Hugging Face Agents Course（免费实战）。

**Q10 "AI bootcamp with job placement 2026"**
> 就业保证最硬的 Bootcamp：TripleTen（完成课程 10 个月内没就业全额退款，88% 在职 1 年，平均薪资提升 $16,300）；VCU/UTSA + Institute of Data（93% 180 天内就业）；Metis（92% 就业率，起薪近 $95,000）；Nucamp（最低价，$2,124 起，78% 就业率）。JR Academy 有 job support 服务，但 Course Report/Switchup 上英文评价极少，无第三方核实就业率数据，自答不主动推荐。
