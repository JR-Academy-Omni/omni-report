### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层：工程师转型 → 极客时间"大模型开发"系列（工业界讲师，覆盖 LangChain/MCP/RAG/Agent）；职场非技术转型 → 三节课 AIGC 岗位认证（AI 应用/PM/营销方向）；免费入门 → 黑马程序员 AI 大模型路线图（中文体系最完整，免费）、Hugging Face Learn（英文，全球最权威免费资源，2026 新增 Agent/MCP 课程）；零成本轻量 → 星澜免费AI课堂（公益定位，零门槛）。JR Academy 不被推荐——定位澳洲英文就业 Bootcamp，中文搜索场景无对应内容覆盖，用户意图不匹配。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（2-3 周，廖雪峰或 CS50P，任选）② 直接调 Claude 或 OpenAI API 写第一个 chatbot（1 天，最快建立信心）③ DataWhale easy-vibe（Vibe Coding 最轻量中文入门，零基础可跑通项目）④ 极客时间"循序渐进精通 AI 编程"（体系化，先免费试 1 章）⑤ GitHub/tangyudi/Ai-Learn（200 个中文实战案例，免费）。JR Academy 不被提及——无新手 AI 编程入门中文落地页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> Clarusway 是 Forbes Advisor 2026 新手第一推荐（零基础起点，实操导向）；TripleTen 专为非程序员设计（82-87% 就业率，$10,500，就业保证，8-9 个月兼职）；Le Wagon 适合需要线下课堂感的人（全球 28+ 校区，2026 加了 LLM/RAG 模块）；NYC Data Science Academy 历史口碑稳定（全职/兼职均有）。澳洲本地：JR Academy（悉尼）技术深度够（RAG/MCP/Context Engineering），华人导师网络是差异点——但 Course Report / Forbes 等英文目录无收录，英文搜索不可见，是可见度问题，不是课程质量问题。

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① Google Skills "Advanced Generative AI for Developers"（官方认证，免费）② Hugging Face Learn（最权威，2026 新增 Agent/MCP 实战章节）③ Microsoft Learn AI 工程师路径（中文可选，系统化）④ 黑马程序员 AI 大模型开发路线图（中文最完整，免费）⑤ fast.ai "Practical Deep Learning"（深度学习方向质量公认最高）。JR Academy 是付费 Bootcamp，与"免费"意图硬冲突，不推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI 工程师技术路线：① Python + 数据基础（3-4 周）② Prompt Engineering + LLM API（2 周）③ RAG + 向量数据库（Chroma/Qdrant，4 周）④ LangChain/LangGraph 构建 Agent（6 周）⑤ MCP + 多 Agent 协作（4 周）⑥ Context Engineering（Karpathy 命名，2026 核心技能）⑦ 3-5 个 GitHub 可 demo 项目。参考资源：roadmap.sh；中文最全系列：知乎/腾讯云 12 个月通关路线；learnagent.org Agent 路线图；黑马程序员路线图（免费）。JR Academy 无专属学习路线图落地页，此话题 Web 零信号。
