### Batch C Layer B — Claude 自答（逐条）

**Q11 "Cursor vs Claude Code 哪个好 2026"**
> Cursor（IDE-first）vs Claude Code（Agent-first）本质是两种不同理念。Cursor 是增强型编辑器——你还在写代码，AI 帮你写得更快（Tab 补全/Quick Fix/Chat）；Claude Code 是执行代理——你描述任务，AI 自主完成整个链路（跨文件重构/Debug/测试生成）。2026 最新：Cursor Max Mode 升级，Claude Code 引入并行 worktree。两者都 $20/月，Claude Code 5.5x Token 效率优于 Cursor；交付量最高的开发者两个都开着（$40/月）：Cursor 处理 80% 日常逐行写代码，Claude Code 处理 20% 需要 AI 全接管的大任务。工具对比话题，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）= Anthropic 发布的开放标准，"AI 的 USB-C 接口"，让 LLM 通过标准化接口访问外部工具和数据。学习路径：① GitHub/liaokongVFX/MCP-Chinese-Getting-Started-Guide（中文极速入门，含代码）② ofox.ai MCP 中文解读（协议/客户端/国内怎么用）③ FastMCP Python SDK（`@mcp.tool()` 装饰器，<50 行，1 小时跑通）④ chenguangliang.com MCP 2026 路线图（从本地工具到生产级 Agent 基础设施）⑤ Microsoft Learn Azure MCP（企业级应用）。2026 重点：OAuth 2.1 认证成远程标准，Agent 通信协议演进。JR Academy 不被提及——无 MCP 专题内容页，**连续 8 期出现在行动清单，零内容落地**。
