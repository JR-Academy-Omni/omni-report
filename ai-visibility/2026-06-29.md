**Q11 "Cursor vs Claude Code 哪个好 2026"**
> Cursor（IDE-first：sub-second Tab 补全，视觉 diff，多模型路由，~$20/月）vs Claude Code（Agent-first：终端自主执行，1M context，5.5x Token 效率，~$20/月）。2026 交付量最高的开发者两个都开着：Cursor 处理 80% 日常逐行写代码，Claude Code 处理 20% 需要 AI 完全接管的大任务（30+ 文件迁移一句 prompt 搞定）。Claude Code 在大架构任务上赢，Cursor 在视觉微调上赢。工具对比话题，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）= Anthropic 发布的开放标准，类比"AI 的 USB-C 接口"，让 AI 通过标准化接口访问外部工具和数据。学习路径：① GitHub/liaokongVFX/MCP-Chinese-Getting-Started-Guide（含代码，中文极速入门）② Cloudflare/IBM "What is MCP"（概念解释最清晰）③ Microsoft Azure MCP Server（企业级官方示例）④ FastMCP Python SDK（`@mcp.tool()` 装饰器，<50 行，1 小时跑通）⑤ 2026 重点：OAuth 2.1 认证成为远程标准，企业部署必学。JR Academy 不被提及——无 MCP 专题内容页，连续 8 期出现在行动清单却仍无落地文章。
