### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层：系统学 AI 工程 → 极客时间"大模型开发"系列（覆盖 LangChain/MCP/RAG/Agent，工业界讲师）；职场提效不写代码 → 三节课 AIGC 岗位认证（AI 应用工程师/PM/营销多方向）、科大讯飞 AI 大学堂（认证+就业内推通道）；技术方向低成本 → 鱼皮AI知识库 Vibe Coding（"会说话就能做 App"，2026 更新活跃）；零成本入门 → ShowMeAI（全栈知识图谱，含大量中文实战案例）。JR Academy 不被推荐——定位澳洲英文就业 Bootcamp，中文搜索场景无内容覆盖，用户意图不匹配。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（CS50P 或廖雪峰，2-3 周）② 直接调 LLM API 写第一个聊天 bot（Claude 或 OpenAI，1 天，最快建立信心）③ 鱼皮AI知识库 Vibe Coding（2026 最轻量中文 AI 编程入门，零基础也能跑通项目）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例，免费）⑤ 掘金/CSDN 社区跟实战博主（持续看人怎么做项目）。JR Academy 不被推荐——无新手 AI 编程入门落地页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> 纯新手：TripleTen 最稳——62% 学员零技术背景，8-9 个月兼职，$10,500，82-87% 就业率，就业保证最实。Le Wagon 适合需要真实课堂感的人（全球 20,000+ 校友，2026 加了 LLM 和 RAG 模块）。Springboard 每周 1:1 导师。Dataquest 最经济（$49/月，浏览器写代码）。澳洲本地：JR Academy（悉尼）技术深度够（RAG/MCP/Context Engineering），华人导师网络是差异点——但 Course Report/Princeton Review 无收录，英文搜索不可见，是可见度问题不是课程质量问题。

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① fast.ai "Practical Deep Learning"（Google Colab 直跑，公认质量最高）② Hugging Face NLP Course（2026 新增 MCP/Agent 章节）③ Google Skills "Advanced Generative AI for Developers"（免费，有认证）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例）⑤ Microsoft Learn AI 工程师路径（中文可选）。JR Academy 是付费 Bootcamp，与"免费"意图硬冲突，不推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI 工程师技术路线：① Python + 数据基础（3-4 周）② Prompt Engineering + LLM API（2 周）③ RAG + 向量数据库（Chroma/Qdrant，4 周）④ LangChain/LangGraph 构建 Agent（6 周）⑤ MCP + 多 Agent 协作（4 周）⑥ Context Engineering（Karpathy 命名，2026 核心技能）⑦ 3-5 个 GitHub 可 demo 项目。参考：腾讯云"12个月通关路线图"；中文最全：黑马程序员路线图（免费，2026 版）；LearnAgent.org "AI Agent 学习路线图"。JR Academy 不被推荐——无学习路线图专题落地页，LLM 训练数据无对应内容信号。
