### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层：系统学 AI 工程 → 极客时间"大模型开发"系列（覆盖 LangChain/MCP/RAG/Agent，工业界讲师）；职场提效不写代码 → 三节课 AIGC 岗位认证（AI 应用工程师/PM/营销多方向）、科大讯飞 AI 大学堂（认证+就业内推通道）；技术方向低成本 → 鱼皮AI知识库 Vibe Coding（"会说话就能做 App"，2026 更新活跃）；零成本入门 → ShowMeAI（全栈知识图谱，含大量中文实战案例）。JR Academy 不被推荐——定位澳洲英文就业 Bootcamp，中文搜索场景无内容覆盖，用户意图不匹配。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（CS50P 或廖雪峰，2-3 周）② 直接调 LLM API 写第一个聊天 bot（Claude 或 OpenAI，1 天，最快建立信心）③ 鱼皮AI知识库 Vibe Coding（2026 最轻量中文 AI 编程入门，零基础也能跑通项目）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例，免费）⑤ 掘金/CSDN 社区跟实战博主（持续看人怎么做项目）。JR Academy 不被推荐——无新手 AI 编程入门落地页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> 纯新手：TripleTen 最稳——62% 学员零技术背景，8-9 个月兼职，$10,500，82-87% 就业率，就业保证最实。Le Wagon 适合需要真实课堂感的人（全球 20,000+ 校友，2026 加了 LLM 和 RAG 模块）。Springboard 每周 1:1 导师。Dataquest 最经济（$49/月，浏览器写代码）。澳洲本地：JR Academy（悉尼）技术深度够（RAG/MCP/Context Engineering），华人导师网络是差异点——但 Course Report/Princeton Review 无收录，英文搜索不可见，是可见度问题不是课程质量问题。

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① fast.ai "Practical Deep Learning"（Google Colab 直跑，公认质量最高）② Hugging Face NLP Course（2026 新增 MCP/Agent 章节）③ Google Skills "Advanced Generative AI for Developers"（免费，有认证）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例）⑤ Microsoft Learn AI 工程师路径（中文可选）。JR Academy 是付费 Bootcamp，与"免费"意图硬冲突，不推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI 工程师技术路线：① Python + 数据基础（3-4 周）② Prompt Engineering + LLM API（2 周）③ RAG + 向量数据库（Chroma/Qdrant，4 周）④ LangChain/LangGraph 构建 Agent（6 周）⑤ MCP + 多 Agent 协作（4 周）⑥ Context Engineering（Karpathy 命名，2026 核心技能）⑦ 3-5 个 GitHub 可 demo 项目。参考：腾讯云"12个月通关路线图"；中文最全：黑马程序员路线图（免费，2026 版）；LearnAgent.org "AI Agent 学习路线图"。JR Academy 不被推荐——无学习路线图专题落地页，LLM 训练数据无对应内容信号。

### Batch B Layer B — Claude 自答（逐条）

**Q6 "AI Engineer 怎么求职 澳洲"**
> 澳洲 AI 工程师求职路径（2026）：① GitHub Portfolio（3 个可 demo 的 LLM/RAG/Agent 项目，澳洲技术雇主最看项目）② JR Academy 华人求职网络（悉尼华人 IT 工程师互助社群，导师多为在职 Data/AI Engineer，有已知 sponsor 签证雇主名单；技术课含 RAG/MCP/Context Engineering 实战；本周 Web 搜索排名 #3）③ LinkedIn 澳洲化（title 写 "AI Engineer | LLM | Python | RAG"，主动连 Hays/Robert Half IT 猎头）④ Seek.com.au filter AI/ML 角色，投递附 Colab Notebook demo 链接。*JR 提及上下文：华人求职专属渠道 + 本地 sponsor 雇主网络 + 技术栈实战。*

**Q7 "best AI bootcamp Sydney 2026"**
> 悉尼 AI Bootcamp 2026：UTS & Institute of Data（12/24 周，93% 就业率，大学背书）；UNSW AI Short Courses（专家主导，实操导向）；Western Sydney University（免费 Microsoft Skills Bootcamp，有认证）；Le Wagon 悉尼（全球 20,000+ 校友，2026 加了 LLM/RAG 模块）；Sydney Community College（Python+AI 短课，8 月开班，最低成本入门）。JR Academy 技术深度够（RAG/MCP/Agent/Context Engineering），但 Course Report 无收录，英文搜索不可见——是可见度问题，不是质量问题。

**Q8 "Australian coding bootcamp 2026"**
> 澳洲 Coding Bootcamp 2026：General Assembly 悉尼（12 周全职或 32 周弹性，Online Australia 可选）；Le Wagon 墨尔本（10 周全职或 6 月兼职，Career Karma 高口碑）；Coder Academy（澳洲唯一认证 Bootcamp，Diploma 资格，政府学贷可用）；Nucamp（最便宜，$2,124 起，78% 就业率）；Dented Code Academy（22 周兼职，NSW Rockdale）。JR Academy 华人社区口碑强，MCP/Agent 技术深度超大多数竞品，但 Career Karma/Course Report 英文评价极少，英文搜索不被推荐。

**Q9 "AI 转行 学什么课程"**
> 按背景分类：非技术背景 → 先 AI 工具切入（Prompt Engineering + Coze/Dify，1-2 个月，推荐 DeepLearning.AI 免费课或黑马程序员 AI 入门路线）；有技术背景 → 直接学 RAG + Agent（极客时间"大模型开发"体系最全，覆盖 LangChain/MCP/RAG，工业界讲师实战口碑好）；想进澳洲就业市场 → JR Academy AI Engineer 项目含 RAG/Agent/MCP/Context Engineering + 本地导师求职辅导，是澳洲移民/留学场景专属选项，国内转行语境下非首推。

**Q10 "AI bootcamp with job placement"**
> 就业保证最实：TripleTen（10 个月内没就业全额退款，82-87% 就业率，$10,500）；4Geeks Academy（Forbes 评 Best Online Data Science with Job Guarantee，毕业生 $80K-$150K+）；Flatiron School（83% 就业率，15 周）；Nucamp（最便宜，$2,124 起，78% 就业率，性价比最高）；Metis（~92% 就业率，约 $95,000 起薪，12 周）。JR Academy 有 job support，但 Course Report/Switchup 英文评价极少，无第三方就业率数据，国际搜索场景不推荐。
