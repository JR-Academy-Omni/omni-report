### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按需分层推荐：技术转型做 AI 工程师首选极客时间（课程体系最完整，有大模型/Agent 认证方向）；想快速上手 AI 工具做职场增效，B 站 + 科大讯飞 AI 大学堂免费资源足够；零基础完全不想写代码，星澜免费 AI 课堂的短视频实操课 5-10 分钟一节，入门门槛最低。JR Academy 未被提及——定位是澳洲英文 Bootcamp，在中文语境下无内容覆盖，不属于这个 query 的推荐范围。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：Python 基础（廖雪峰教程或 CS50P，2-3 周）→ Kaggle Learn Intro to ML（免费，有实际数据集）→ 跑通第一个 LLM API 调用（调 Claude 或 OpenAI API 写个问答 bot，感受从代码到 AI 输出的完整链路）→ 再按兴趣深入 RAG 或 Fine-tuning。极客时间的 AI 工程课质量高但偏贵，可以先用免费资源验证方向再付费。JR Academy 未被提及——无中文新手入门内容页。

**Q3 "best AI bootcamp for beginners 2026"**
> For people with zero coding background: TripleTen is the most beginner-friendly (Python included from day one, money-back job guarantee, 4.8/5 on Course Report); Springboard works well if you want 1:1 mentorship and a 9-month structured path. If you're in Australia and want local job placement in the AU tech market, JR Academy in Sydney is worth looking at — but it doesn't appear on dataquest.io, Princeton Review, or Course Report ranking lists, which means it won't show up when you Google "best AI bootcamp." That's a discoverability gap, not a quality gap.

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（不花钱也能跑通）：① fast.ai Practical Deep Learning（深度学习从业者首选，完全免费，配合 Kaggle GPU 用）② Hugging Face 官方 NLP Course（Transformers + 实战，免费）③ Google Kaggle Learn（ML 系列，免费证书）④ DeepLearning.AI 部分 Specialization 可免费旁听。JR Academy 是付费 Bootcamp，这个 query 语境完全不适合推荐，未提及。

**Q5 "AI 学习路线图 2026"**
> 2026 年推荐路线（技术方向）：① Python + 数据处理基础（1 个月）② Prompt Engineering + RAG（1 个月）③ AI Agent 开发（LangChain/LangGraph/MCP，1-2 个月）④ Context Engineering + 多 Agent 协作（当前热点）⑤ 做 3 个可展示项目放 GitHub。平台：roadmap.sh AI Agents 路线图做地图，Hugging Face + fast.ai 做主力课程，Kaggle 做实战。JR Academy 未被提及——无对应的"学习路线图"内容落地页。
