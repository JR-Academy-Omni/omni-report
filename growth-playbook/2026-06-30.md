| **预估指标** | 今天 1 天内锁定 6–8 家企业询盘，转化 3 家（每家 4 人 × $2,600 折后）= **$31,200 AUD pipeline**；EOFY 后给未成交线索发「FY27 Q1 早鸟价」续跟进 |

4. **晚上 11:59**：名额关闭，发「名额已满」宣布帖；明天给所有提交但未成交的线索发「FY27 Q1 早鸟价」邀约

## 玩法 ② **Copilot 账单危机 × AI 编程工具省钱计算器**（工具钩子 + 转化 AI Engineer）

| 字段 | 值 |
|---|---|
| **一句话描述** | GitHub Copilot 6月1日起改按量计费（$0.01/credit），中重度开发者月均成本超预期——JR 上线一个「AI 编程工具月成本计算器」（填每天用几次 Copilot / Cursor / Claude Code，自动估算月费），结果页给出最省钱的 3 种组合方案，方案 C 指向 AI Engineer Bootcamp（学会用 Claude Code 免费方案替代 $30/月 订阅）|
| **灵感来源** | HubSpot Website Grader（免费诊断工具 → 付费漏斗）+ 知乎答题「算完觉得亏了」心理激活 |
| **对应业务** | AI Engineer Bootcamp（拉新 + 付费转化）+ Prompt Lab（激活试用）|
| **目标用户** | 拉新——收到 Copilot 账单超预期的在职开发者；激活——已注册 Prompt Lab 但没系统学过 Claude Code 的用户 |
| **执行难度** | 🟢 小（Typeform 或 Tally 表单 + 条件逻辑显示 3 个结果页文案；无后端开发，1 天上线）|
| **预估指标** | 1 周完成 800 次计算；邮件捕捉 600 人；「选方案 C」率 35% = 210 人进入 AI Engineer Bootcamp 漏斗；付费转化 18 人 × $2,500 = **$45,000 AUD pipeline** |
| **本周做？** | ✅ 推荐本周启动（Copilot 计费改变刚发生 2 周内是开发者讨论最热的窗口；下个月大家「接受现实」后焦虑感消失，工具的触发力减弱）|

**落地步骤**：
1. **D0**：用 Tally 搭 3 题计算器（Q1：你每天用 AI 编程工具几小时？1h / 2h / 4h+ / 仅试用；Q2：你目前付费用哪些？Copilot / Cursor / Claude Pro / 全部免费；Q3：填邮件，看你的省钱方案）；设计 3 个结果页：方案 A（轻度用户，现有免费层够用，无需付费）、方案 B（中度用户，Cursor Hobby + 免费 Claude Code 组合，月省 $20+）、方案 C（重度用户，系统学 Claude Code 本地模式 + 自建 API 调用，月均成本降到 $8 以下）；方案 C 末尾加「这是 AI Engineer Bootcamp 第 3 周课程教的核心工具技巧 → 查看课程」
2. **D1**：LinkedIn / X 发帖「我算了一下：一个中度开发者用 GitHub Copilot 新计费方案，月均 $47。我整理了 3 种省钱组合，帮你算清楚」；附计算器链接；B 站短视频「你的 Copilot 账单算对了吗」（2 分钟，屏幕录制走一遍计算器）
3. **D2–D5**：运营在开发者微信群 / Discord 分享计算器链接；每天在 LinkedIn 评论区回复 1–2 条「我测了，结果是方案 B」的自然讨论，拉互动
4. **D3**：给选了方案 C 的用户发 Email Sequence 第 1 封「你的结果：每月可以省下 $XX。这是 AI Engineer Bootcamp 里教的第一个省钱步骤（附一段截图）」；3 天后发第 2 封「方案 C 的学员里，有多少人已经在用」
5. **D7+**：汇总数据，把「最多人选择哪种方案」做成一条 LinkedIn 结果帖「我们收集了 800 份开发者数据：43% 的人 Copilot 新计费方案其实在多花钱」

## 玩法 ④ **AI PM「真人感」案例周：出镜展示你 AI 改造的真实产品**（UGC 内容 + 付费转化）

| 字段 | 值 |
|---|---|
| **一句话描述** | 小红书/B站算法本周正在推「真人感」内容（AI 生成泛滥后，真实出镜 > 精美排版）——JR 发起「AI PM 真人周」：招募 5 名 AI PM Bootcamp 学员或 PM 职场人，每人出镜 60 秒讲清「我用 Claude / GPT-5.4 重新设计了 [产品] 的 [一个功能]，PRD 从 2 天写到 2 小时」，附真实 Notion PRD 截图，发小红书 + B 站；完成发布的参与者获得 AI PM Bootcamp 1 周免费试学 |
| **灵感来源** | 元気森林 UGC 内容机制（用户真实体验 > 品牌自我声称）+ 小红书「出镜真实感」当前算法偏好 |
| **对应业务** | AI PM Bootcamp（拉新 + 付费转化）|
| **目标用户** | 拉新——小红书上搜「AI PM 怎么用」「AI 写 PRD」的职场 PM；付费转化——看到真实案例后觉得「我也能做」的观望者 |
| **执行难度** | 🟢 小（Discord 学员群招募 5 人 + 给拍摄指南 + 审核发布；无技术开发，运营 1 天可启动）|
| **预估指标** | 5 帖合计播放 15 万次（「真人感」类内容当前小红书单帖均值 3 万）；涨粉 400；PRD 模板下载 500 次；AI PM Bootcamp 报名增加 35 人；付费转化 8 人 × $1,800 = **$14,400 AUD** |
| **本周做？** | ✅ 推荐本周启动（小红书「活人感」算法窗口是当前 3–4 周的短期红利；GPT-5.4 computer-use 刚发布，AI PM 话题正在热搜，出镜讲「AI 改变了我写 PRD 的方式」有自然流量加成）|

**落地步骤**：
1. **D0**：Discord 学员群发招募帖「我们在找 5 名愿意出镜 60 秒的 AI PM / PM 职场人，讲你用 AI 工具重新设计产品功能的真实案例，完成发布得 AI PM Bootcamp 1 周免费试学 + JR 官号转发」；筛选标准：有真实产品经验（在职 PM 或学员）+ 有可展示的 PRD / 原型截图
2. **D1**：给 5 名参与者发拍摄指南（60 秒结构：0–10s 开门见山「我在 [公司/职位]，我用 Claude 重新设计了 [产品] 的搜索功能」/ 10–50s 展示 Notion PRD 截图 + 一句话说改造前后差异 / 50–60s 「这是 AI PM Bootcamp 教的方法，链接在 Bio」；要求：真实背景出镜，不要 AI 生成封面，标题禁「最强/颠覆」等词）
3. **D2–D5**：每天 1 帖按序发布；JR 官方账号在每帖发布后 30 分钟内转发 + 补一条「这是我们学员的真实 PRD，不是模板」；每帖附「评论区领 AI PM 提示词包」（评论「PRD」→ 私信发资料包，激活评论互动提升算法权重）
4. **D5（所有帖发完后）**：将 5 帖整合成「5 位 PM 如何用 AI 把 PRD 时间缩短 80%」Notion 合集页，发 B 站 5 分钟剪辑；给从帖子点进来的新用户自动发「PRD 模板包 + AI PM Bootcamp 48 小时限时试看」邮件
5. **D6+**：对 5 位参与者追加采访「发帖之后有哪些 PM 来问你」，这批反馈直接作为下期 AI PM Bootcamp 招生页社会证明
