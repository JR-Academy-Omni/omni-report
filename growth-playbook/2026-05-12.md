## 玩法 ① **DeepLearning.AI 学完了？JR 带你做真项目：AG-UI 接力工作坊**（竞品反向钩）

| 字段 | 值 |
|---|---|
| **一句话描述** | 上传 DeepLearning.AI「Build Interactive Agents with Generative UI」完课截图，自动解锁 JR 限额 50 名「AG-UI × Claude Code 一天工作坊」免费席位（5/16 线上），交付物：一个真实 SaaS 场景的 AI Agent 前端，可直接上 GitHub 作品集 |
| **灵感来源** | GitHub Student Pack（身份验证解锁权益）+ 接力课程漏斗（免费入门课 → 付费进阶实战） |
| **对应业务** | AI Engineer Bootcamp（拉新 + 付费转化） |
| **目标用户** | 拉新——正在学 DeepLearning.AI 的工程师，已被验证有自学意愿，缺实战项目经验 |
| **执行难度** | 🟡 中（1 天工作坊内容 + 上传验证落地页 + Zapier 发席位邮件，3 天可搭通） |
| **预估指标** | 1 周：800 人点击落地页，50 人提交完课截图进工作坊，工作坊后 40% 预约 AI Engineer Bootcamp 试听 = 20 人，8% 转化付费 = 4 人高质量报名 |
| **本周做？** | ✅ 推荐本周启动（DeepLearning.AI 课 5/6 上线，热度窗口 2-3 周；工作坊定 5/16 周六，今天启动筹备刚好 3 天；再晚话题降温） |

**落地步骤**：
1. **D0（今天）**：教研负责人确认 5/16 工作坊 outline——AG-UI 协议配置（1.5h）→ Claude Code + MCP 工具链接入（1.5h）→ 真实 SaaS 场景实战并 deploy（2h）；找好 1 位有 AG-UI 生产经验的讲师或助教
2. **D1**：搭落地页（Webflow 1 页：接力计划介绍 + 上传完课截图 Typeform + 50 席位实时倒计时）；Zapier 配置：截图提交 → 自动发确认邮件 + Google Calendar 邀请
3. **D1**：讲师在 LinkedIn 发文「DeepLearning.AI 的 AG-UI 课不教你的 3 件事」——正文指出该课缺 MCP 集成、无真实部署环境、不涉及多模型切换；文末插接力计划链接
4. **D2**：在 DeepLearning.AI 社区公告帖下留有价值的技术补充评论（不做广告），附带工作坊链接；同步在 JR 学员群推荐给已学完 AG-UI 相关内容的学员
5. **D5（5/16）**：工作坊收尾时收集所有参与者 GitHub 项目链接，次日发布「50 名工程师 5 小时产出的 AG-UI Agent 合集」文章（知乎 + LinkedIn），作为长尾引流持续 3 个月

## 玩法 ② **AI 合规就绪度测试：你的企业 AI 部署能过白宫新标准吗？**（B2B 内容钩子）

| 字段 | 值 |
|---|---|
| **一句话描述** | 5 分钟问卷（15 道题：AI 应用范围 × 数据隐私措施 × 员工 AI 培训覆盖率），自动生成「AI 合规就绪度」0-100 分报告 + 3 个具体高危风险项；报告一键转发给 HR/CTO；报告末尾 CTA：预约 JR AI Adoption Bootcamp 企业版 30 分钟 Demo，前 20 家企业免费做 AI 合规现状评估 |
| **灵感来源** | HubSpot 网站评分工具（自测钩子）+ B2B 内容营销（政策热点 → 企业 leads） |
| **对应业务** | AI Adoption Bootcamp（企业团购 B2B）|
| **目标用户** | 付费转化——澳洲 100-500 人规模企业的 IT 负责人 / HR Director / Operations Manager |
| **执行难度** | 🟡 中（Typeform 问卷 + 评分逻辑 + Claude API 生成个性化风险报告 + WeasyPrint 转 PDF + Zapier，2-3 天可搭通） |
| **预估指标** | 2 周：300 家企业完成测评，25% 预约 Demo = 75 个企业 leads，15% 成交企业培训 = 11 家，均价 $5,000 AUD = $55,000 AUD pipeline |
| **本周做？** | ✅ 推荐本周启动（白宫 FDA 式 AI 审查 EO 5/7 曝光；澳洲政府 AI 安全框架同期讨论中；企业 IT 部门正在问"合规该怎么做"，这是 1 个月内最强政策钩子） |

**落地步骤**：
1. **D0**：制定 15 道测评题——分三组：AI 应用范围（哪些部门在用 / 用什么工具）× 数据安全（数据是否出境 / 有无访问控制）× 培训覆盖率（多少员工受过 AI 培训 / 有无培训记录）；各组 0-33 分，合计 100 分；< 60 分标红"高风险"
2. **D1**：Typeform 搭问卷 + Zapier：提交后 → Claude API 生成 3 个具体高危项描述（基于作答内容，不是模板）→ WeasyPrint 打包为 PDF → 自动邮件发送；报告内嵌"一键转发给 HR/CTO"按钮（mailto 链接，预填标题 + 正文）
3. **D2**：AI Adoption Bootcamp 负责人在 LinkedIn 发文「澳洲企业 AI 合规：白宫新标准来了，你的业务准备好了吗？」——内嵌 3 个真实高风险场景（如：员工用 ChatGPT 处理客户数据但无书面政策），末尾插自测链接
4. **D3**：向 LinkedIn 筛出的 100 位澳洲 HR Director / IT Manager 发冷邮件（Instantly.ai / Apollo），主题「3 分钟测你的企业 AI 合规风险」，邮件正文 5 句话 + 链接，不发任何课程广告
5. **D7**：整理前 100 家完成测评的企业数据（脱敏聚合），发布「澳洲中小企业 AI 合规就绪度调研 2026 Q2」PDF 报告，发 AFR / AusBiz 媒体联系，作为 PR 材料长期引流

## 玩法 ③ **Routines Builder Sprint：7 天用 Claude Code Routines 自动化你的副业**（streak + 裂变）

| 字段 | 值 |
|---|---|
| **一句话描述** | 报名后每天收到 1 个 Claude Code Routine 构建任务（D1: 自动 PR review → D2: 代码质量检查 → D3: 周报生成 → D4: 客服回复 → D5: 测试用例生成 → D6: 文档同步 → D7: 全链路集成），每天提交 GitHub commit 截图到 JR 社区打卡；完成 7 天领 Vibe Coding Bootcamp $300 off；邀请 1 位朋友同期打卡 → 双方额外得 $100 credit |
| **灵感来源** | Duolingo streak（每日打卡 + 连续感）+ Dropbox 邀请翻倍（双向裂变）|
| **对应业务** | Vibe Coding Bootcamp（激活 + 拉新）|
| **目标用户** | 激活（已注册 JR 未付费学员）+ 拉新（Claude Code 重度用户 / 正在试用 Routines 的工程师）|
| **执行难度** | 🟡 中（7 天任务包 + 社区打卡频道 + 打卡追踪 spreadsheet；教材可复用现有 Vibe Coding 内容，主要工作量在任务文案和打卡流程配置）|
| **预估指标** | 1 周报名：400 人；7 天完成率 60% = 240 人领 $300 off；25% 转化付费 Vibe Coding = 60 人；邀请裂变带来 80 新参与者，20% 转化 = 16 人付费 |
| **本周做？** | ✅ 推荐本周启动（Claude Code Routines 5/6 公测，JR 有机会成为第一批出中文 Routines 实战教程的机构，2 周后竞品跟进） |

**落地步骤**：
1. **D0（今天）**：教研团队敲定 7 天任务包——每天 1 个 Routine，预计 30-45 分钟完成，给代码框架 + 验收标准（不是空题，有脚手架）；在 Discord / 微信群开「Routines Sprint 打卡频道」
2. **D1**：发报名落地页（核心文案「Claude Code Routines 公测来了，7 天把副业自动化，交一份作品集」）；报名后自动发 D1 任务包邮件 + 带 referral UTM 的邀请好友链接
3. **D1**：在小红书 / B站发 60 秒短视频「Claude Code Routines 是什么？三句话 + 我们 7 天挑战计划」——不做教程，只做钩子；同步发 JR 学员群引流
4. **D2–D6**：每天发 2 分钟当日任务讲解录屏（讲师或助教录，不需要精剪），社区内评选当日最佳提交，给 $20 虚拟 credit，制造 FOMO
5. **D7**：直播展示 Top 3 Routine 代码，上传到 B站作为常青内容；同日向所有完成打卡者推送「你已经掌握了 Routines，Vibe Coding Bootcamp 会教你用它做一个完整 SaaS」转化邮件

## 玩法 ④ **15分钟 AI PRD 闪电赛：Claude 帮你写，PM 社区来评**（计时挑战 + UGC）

| 字段 | 值 |
|---|---|
| **一句话描述** | 系统每周给出 1 个真实 AI 产品场景（本周：为 50 人 SME 设计一个中英双语 AI 客服 Agent，需与 HubSpot CRM 对接），参与者用 Claude 在 15 分钟内完成 PRD 第一版并提交；JR AI PM 讲师盲评 Top 10，Top 10 脱敏后发布 LinkedIn；作者得 AI PM Bootcamp $200 credit；所有参与者得「Claude PM 工具包」（5 个 Prompt 模板 + PRD 框架 PDF）|
| **灵感来源** | LinkedIn 技能徽章（PM 社区可见性驱动）+ 知乎答题瓜分（有限名额的竞争感）|
| **对应业务** | AI PM Bootcamp（拉新 + 付费转化）|
| **目标用户** | 拉新——在职 PM / 想转型 AI PM 的职场人，LinkedIn 澳洲 PM 社区 |
| **执行难度** | 🟢 小（场景出题 + Typeform 提交 + 人工盲评 + 邮件发奖；无需技术开发，2 天可上线）|
| **预估指标** | 1 周：300 人参与，全部得工具包（= 300 个有效邮件地址），Top 10 发 LinkedIn 后 2,000+ 曝光，10% 预约 AI PM Bootcamp 试听 = 30 人，20% 转化付费 = 6 人 |
| **本周做？** | ✅ 推荐本周启动（Anthropic ARR 超越 OpenAI 是「为什么选 Claude 做 PM 工具」的最强时机背书；5/12 启动 → 5/15 关闭提交 → 5/16 出 Top 10，节奏紧凑）|

**落地步骤**：
1. **D0（今天）**：确定本周题目（「为 50 人规模澳洲 SME 设计 AI 客服 Agent：支持中英双语、与 HubSpot CRM 对接、处理退款 / 订单查询 / 常见问题三类请求」）；准备盲评评分维度（问题定义清晰度 × 用户故事完整性 × 功能范围合理性 × 成功指标可衡量性，各 25 分）
2. **D1**：Typeform 搭提交表单（姓名 + 邮件 + 职业背景 + 贴 PRD 文本或 Notion 链接）；落地页核心文案「你有 15 分钟和 Claude，能写出一份合格的 AI 产品 PRD 吗？」；在澳洲 Product Manager Community（LinkedIn 群组）+ AIIA AI in Products 频道发帖引流
3. **D1–D3**：提交窗口开放（诚信计时制，不强制在线掐表）；每天在 LinkedIn 发 1 条「当前提交数 + 截止倒计时」更新，保持话题热度
4. **D4**：讲师用 3-4 小时完成盲评（10 份 × 20 分钟），批量发 $200 credit 兑换码给 Top 10；给所有提交者发「Claude PM 工具包」邮件（工具包提前准备好）
5. **D5**：Top 10 PRD 脱敏版发 LinkedIn（每篇一个帖子，带「#AIPMChallenge #JRAcademy」标签）；同步发转化邮件给所有参与者：「你已经知道 AI PRD 是怎么写的了，AI PM Bootcamp 会教你把它变成真实产品」

## 玩法 ⑤ **2,206 个坑：JR 学员内推集结令**（双向裂变 + 求职数据钩子）

| 字段 | 值 |
|---|---|
| **一句话描述** | SEEK 本周 2,206 个 AI Engineer 在招岗位、薪资 $165k–$250k；JR 学员每人提交 1 个内推意向（我能推进 XX 公司 AI 团队），JR 做撮合方匹配求职学员；成功入职 → 内推者得 $500 JR Credit，被内推者得 $500 JR Credit；提交 3 个意向 → 额外得「JR 就业数据包」（含 SEEK / LinkedIn 岗位实时追踪 + 薪资谈判模板）；邀请 1 位非学员求职者进系统 → 得 $200 JR Credit |
| **灵感来源** | Dropbox 双向空间奖励（推荐者 + 被推荐者各得）+ 邀请阶梯奖（数量触发不同奖励）|
| **对应业务** | 求职服务 Job Referral + AI Engineer Bootcamp（老学员留存 + 新用户激活）|
| **目标用户** | 留存（老学员重激活）+ 拉新（通过老学员社交网络触达潜在新学员）|
| **执行难度** | 🟡 中（内推意向登记 + Airtable 撮合 + Credit 追踪；MVP 用 Google Sheet 手动撮合，验证参与率后再自动化）|
| **预估指标** | 2 周：150 名学员提交内推意向（目标现有学员 30% 参与），覆盖 80 家企业，成功推进面试 15 人，3 人最终入职；求职服务新增 60 个外部求职者注册（通过邀请链接）|
| **本周做？** | ⏳ 下周启动（SEEK 数据本周新鲜，但撮合系统需 3–4 天搭建；本周先在学员群发意向收集 Google Form 验证参与意愿——若 D3 前有 30 人填，下周一正式上线）|

**落地步骤**：
1. **本周**：在 JR 学员微信群 / Discord 发「内推意向调查」Google Form（公司名 / 能推哪个 AI 相关岗位 / 联系方式），发布时附上 SEEK 2,206 数据截图，目标 3 天内 50 人填写
2. **D0（下周一）**：Airtable 搭内推数据库（公司 × 岗位 × 内推人 × 求职者 × 状态），设计 3 步撮合流程：求职者报名 → JR 运营匹配内推人 → 内推人联系 HR → 反馈结果
3. **D1**：向所有现有求职服务学员发邮件「SEEK 2,206 个 AI Engineer 岗位，JR 内推网络开放」+ 带 referral UTM 的邀请链接（邀请非学员求职者进系统得 $200 credit）
4. **D3**：发「本周内推进展」社区帖（哪位学员帮哪位同学推进了哪家公司，脱敏处理），制造 FOMO + 信任感；非参与学员看到后跟进率预计 +20%
5. **D14**：第一批入职确认后（预计 1–2 人），协助其发 LinkedIn 感谢帖，JR 官号转发 + 标注 Job Referral 服务；这条内容作为 Job Referral 产品长期引流资产
