## 玩法 ① **EOFY 企业 AI 培训闪电窗：28 天用完 or 废掉**（限时 B2B + 冷外展）

| 字段 | 值 |
|---|---|
| **一句话描述** | 6 月 30 日澳洲财年截止，JR 向 150 家 50 人以上企业的 HR/L&D Manager 发起 LinkedIn DM 外展：锁定 EOFY 团购价（3+ 座位 $2,800 AUD/人，原价 $3,200），6 月 30 日前完成付款，AI Adoption Bootcamp FY26 培训预算合规计入 |
| **灵感来源** | 618 早鸟价（明确截止 + 价格特权）+ HubSpot B2B 冷外展漏斗（LinkedIn DM → Zoom → 合同） |
| **对应业务** | AI Adoption Bootcamp（企业版，非技术职场人团购） |
| **目标用户** | 付费转化——澳洲 50 人以上企业的 Training Manager / HR Manager / L&D Director，FY26 培训预算 6 月 30 日到期 |
| **执行难度** | 🟡 中（EOFY 落地页 + 企业报价单 PDF + LinkedIn 外展序列；无需技术开发，2 天可备好） |
| **预估指标** | 接触 150 家企业，30 家回复，12 家 Zoom 咨询，6 家成交（均 4 人座位）= 24 席 × $2,800 = **$67,200 AUD pipeline**；已有企业客户复购率目标 30% |
| **本周做？** | ✅ 推荐本周启动（6 月 2 日距 EOFY 仅 28 天，外展→Zoom→合同→付款最少需要 1.5 周；D0 不发就来不及） |

**落地步骤**：
1. **D0**：制作 EOFY 企业培训一页纸（AI Adoption Bootcamp 课程大纲 + 阶梯团购定价：3 人 $2,800/人 / 5 人 $2,600/人 / 10 人 $2,400/人，含「FY26 培训预算使用确认函」PDF 供财务存档）；在官网新增 `/eofy-corporate` 落地页（倒计时 Widget + 立即咨询表单 + Calendly 预约 Zoom）
2. **D1**：LinkedIn DM 外展：筛选澳洲 HR Manager / L&D Director / Training Manager（公司规模 50+），发 6 句话消息（「FY26 培训预算 6 月 30 日关闭，我们有 [团队规模] 个 AI 技能团购席位——想聊 15 分钟？」），第一批 100 条
3. **D2**：给 JR 历史企业客户（曾采购过 Bootcamp 的公司）发优先通知邮件「老客户 EOFY 折扣仅限本月，最后 4 周」；WeChat / Slack 行业群同步
4. **D7–D14**：跟进回复但未预约的企业「截止日还有 X 天——您们 FY26 培训预算方向是什么？我可以帮您出一份定制方案」；有意向者发定制版企业 AI 培训方案（列明课程日期、人数弹性、开票方式、合规材料）
5. **D21（6 月 23 日）**：最后 7 天紧迫推送，给所有接触过但未成交企业发「Final call: EOFY 企业 AI 培训最后 7 天锁价」

## 玩法 ② **「Mythos 来之前」7 天 Agent 闯关赛**（streak 挑战 + 热点预热）

| 字段 | 值 |
|---|---|
| **一句话描述** | Anthropic Mythos「数周内」全量开放，JR 先发制人：发起 7 天 Agentic 闯关赛（Day 1: 基础 Claude chat agent → Day 7: 多工具串联可部署 API），每天提交截图上排行榜；全部完赛前 100 名抽 3 个 AI Engineer Bootcamp 免费席位（$2,500 AUD），所有完赛者发 Mythos 优先体验资格码 |
| **灵感来源** | Duolingo 连续 streak（每日完成 + 中断有挫败感）+ B 站 7 天挑战赛（内容自生产）+ Strava 月度挑战（公开排行榜） |
| **对应业务** | AI Engineer Bootcamp |
| **目标用户** | 拉新——用过 Claude/ChatGPT 但没系统做过 Agentic 开发的开发者；激活——已注册 AI Engineer Bootcamp 但还没认真开始的学员 |
| **执行难度** | 🟡 中（7 道任务题（含代码框架）+ Notion 排行榜 + 每日邮件序列 + B 站直播 Demo Day；教研 2 天出题，运营 1 天配邮件） |
| **预估指标** | 1 周报名 1,000 人；完成 Day 1 提交 600 人；完赛（Day 7）200 人；完赛者中 25% 转化 AI Engineer Bootcamp 咨询 = 50 人 × $2,500 AUD = **$125,000 AUD pipeline**；B 站 Demo Day 直播 5,000 在线 |
| **本周做？** | ✅ 推荐本周启动（Mythos「数周内」意味着最多 2–3 周后发布；热度窗口仅这 1–2 周，等 Mythos 真发布再做就没有「预热」意义） |

**落地步骤**：
1. **D0**：教研出 7 道任务（Day 1: 用 Claude chat 写一个抓天气 + 发邮件脚本；Day 2: Claude Code 重构一段 Python 遗留代码；Day 3: 接入外部 API + Claude 做数据分析 agent；Day 4: Claude Code 在真实 GitHub repo 上定位并修复一个 bug；Day 5: 多工具串联——搜索 + Claude + 数据库读写；Day 6: 带错误处理 + 重试机制的自愈 agent；Day 7: 把 Day 5 的 agent 封装成可部署的 FastAPI 接口）；Notion 数据库收提交（显示名 + GitHub 链接 + 提交时间，自动显示排行榜）
2. **D1（周三）**：全平台发起——B 站视频「Anthropic Mythos 要来了，你用 Opus 4.8 能走到第几天？」+ 小红书图文 + LinkedIn/X 英文帖 + 学员 Discord/WeChat 群；Typeform 报名上线（名字 + 邮件 + GitHub ID，提交后立即发 Day 1 任务链接）
3. **D1–D7**：每天 7:30am AEST 发任务邮件（主题「[Day N/7] Mythos 闯关：今天的任务是……」，含昨日完赛人数 + Notion 排行榜链接）；Discord 新建 `#mythos-challenge` 频道，参与者互晒代码，运营每天转发 2 条最有趣提交到小红书
4. **D7**：B 站直播 1 小时 Demo Day——Top 10 完赛者各 3 分钟展示 Day 7 API，直播间观众投票，Top 3 各得 1 个 AI Engineer Bootcamp 免费席位；录屏剪 3 分钟精华版发小红书
5. **D8**：给 200 名完赛者发邮件「你完成了 Mythos 预热挑战——这是你的 AI Engineer 能力档案 + AI Engineer Bootcamp $300 off 折扣码（48 小时有效）」

## 玩法 ③ **SEEK 2491 职位 5 分钟 AI Engineer 匹配度测验**（免费诊断工具 + SEO）

| 字段 | 值 |
|---|---|
| **一句话描述** | JR 在 /learn 页面上线 10 题选择题（覆盖 SEEK 高频 JD 要求：Python / Claude API / RAG / MCP / 系统设计），做完立即显示「你匹配 2,491 个职位中的多少个」（分 4 档）+ 缺什么技能 + 对应推荐（缺技能→AI Engineer Bootcamp，技能够→Job Referral）；结果页一键分享 LinkedIn |
| **灵感来源** | HubSpot Website Grader（免费诊断→付费服务转化漏斗）+ 知乎测专业排名（测完显示百分位引发满足感/焦虑感） |
| **对应业务** | AI Engineer Bootcamp（技能不足者转化）+ 求职服务 Job Referral（技能充足者转化） |
| **目标用户** | 拉新——正在找澳洲 AI Engineer 岗的开发者（搜「澳洲 AI Engineer 要求」）；激活——已注册未付费、不确定自己水平的学员 |
| **执行难度** | 🟢 小（10 道选择题 + Typeform embed + 4 档结果页文案 + LinkedIn 分享按钮；1.5 天可上线，无需后端） |
| **预估指标** | 1 周完成测验 2,000 人，邮件捕捉 1,200 人，AI Engineer Bootcamp 咨询 60 人，Job Referral 预约 30 人；LinkedIn 分享带 300 名新访客；测验页面 3 个月内成为「澳洲 AI Engineer 要求」关键词排名资产 |
| **本周做？** | ✅ 推荐本周启动（SEEK 数据本周更新至 2,491，数字最新鲜；制作难度小，D0-D1 可上线；越早发越早积累 SEO 权重） |

**落地步骤**：
1. **D0**：出 10 题测验（从 SEEK JD 高频词中提取，例：「你用 Claude API 做过哪类集成？A 没用过 B 基础 Chat API C Tool use D Multi-agent 系统」/ 「你独立部署过 RAG 系统吗？A 没听说过 B 知道原理没做过 C 做过 Demo D 做过生产级别」），设计 4 档结果文案（0–25%：「还差 3–4 个核心技能，AI Engineer Bootcamp 6 周补齐」；25–50%：「基础扎实，差工程化经验，Bootcamp 有对应模块」；50–75%：「达到初级岗要求，建议做项目提升后看 Job Referral」；75%+：「已匹配大多数岗位，缺的是简历表达和面试转化——Job Referral 直接帮你投」）
2. **D1**：/learn 页嵌入 Typeform quiz 上线；测验结束后要求填邮件才显示「你缺了哪些技能」详细清单；结果页含「分享我的 AI Engineer 匹配度」LinkedIn 按钮（预填文案：「我测了一下自己和 2491 个澳洲 AI Engineer 职位的匹配度，结果是 X%，链接在评论」）
3. **D2**：LinkedIn 发帖「我们分析了 SEEK 上 2,491 个 AI Engineer JD，用 10 题测出你能匹配几个——结果有点出乎意料」（含 Top 5 高频要求数据，测验链接在评论）；小红书发「澳洲 AI 工程师招聘大数据：你真的够格吗？5 分钟自测」
4. **D3–D7**：在知乎「澳洲 IT 求职」「AI 工程师转型」话题下发回答时引用测验数据（「截至今天 800+ 参与者中，匹配 75%+ 职位的只有 11%」），文末附测验链接
5. **D7+**：给 1,200 个邮件按档位发培养序列（0–25% 档→AI Engineer Bootcamp 学员就业案例；75%+ 档→Job Referral 服务介绍 + 面试成功率数据）

## 玩法 ④ **OpenAI IPO 倒计时：AI PM 工具风险地图**（案例库换反馈 + 公开点评）

| 字段 | 值 |
|---|---|
| **一句话描述** | OpenAI 9 月 IPO 后 API 定价将随季报波动，JR 向 AI PM / 产品负责人发起「工具依赖风险自查」：填写当前 AI 工具栈（工具名 / 月调用量 / 成本 / 替代方案有无），提交后即得「AI 工具风险评级」一页报告 + 多元化决策矩阵 Notion 模板；5 份工具栈被 AI PM Bootcamp 讲师在 Zoom 直播中逐一拆解，被选中者得 AI PM Bootcamp 免费试听 |
| **灵感来源** | Sequoia Scout 报告模式（提交真实数据换专家分析）+ 知乎问答公开讲评（点评制造围观，被点评者有虚荣心） |
| **对应业务** | AI PM Bootcamp（付费转化）+ AI Adoption Bootcamp 企业版（B2B pipeline） |
| **目标用户** | 付费转化——在公司主导 AI 工具采购的 PM / CTO / IT Manager，被 OpenAI IPO 新闻触发「我们是不是太依赖 OpenAI 了」的焦虑；拉新——LinkedIn 关注 AI 工具选型话题的 B 端产品人 |
| **执行难度** | 🟡 中（Notion 模板 1 天出 + Typeform 收表 + 1 场 Zoom 直播 60 分钟；需 AI PM 讲师出镜，宣传靠 LinkedIn 外展和学员群） |
| **预估指标** | 收到工具栈提交 50 份，直播 120 人在线，直播后 80 人下载 Notion 模板，8 人付费 AI PM Bootcamp（$1,800 AUD）= **$14,400 AUD**；B2B 端 3 家公司预约 AI Adoption 企业培训咨询 |
| **本周做？** | ✅ 推荐本周启动（OpenAI 保密 S-1 是 5 月 22 日新闻，本周是企业 PM 讨论热峰；等 2 周 IPO 进度不确定性降低，焦虑驱动消散） |

**落地步骤**：
1. **D0**：出「AI 工具依赖风险自查表」（8 列：工具名 / 用途 / 月成本 USD / 月调用量 / 是否有 SLA / 替代方案 / 切换难度 1–5 / 核心业务依赖 Y/N）；Typeform 上线（提交表格 + 公司规模 + 行业 → 自动发 Notion 决策矩阵模板，含「单一供应商 vs 多元化 TCO 计算器」）
2. **D1**：LinkedIn 发帖「OpenAI 9 月 IPO：你们产品还有多少 API 调用依赖它？」——正文给「IPO 后定价波动 3 个场景」（维持 / 涨 20% / 分层计费），每个场景对应企业应对策略，文末工具栈提交链接；WeChat 产品/IT 管理群转发
3. **D2**：筛选 50 份提交中最典型的 5 份（覆盖不同依赖度：纯 OpenAI / 混合 / 已多元化），联系被选中者获授权；Zoom 直播报名开放（所有提交者 + 公开报名）
4. **D4（直播）**：Zoom 直播「5 份真实 AI 工具栈被逐一风险评级」（每份 10 分钟：读工具栈→评风险点→给多元化路径），直播结束前 20 分钟介绍 AI PM Bootcamp「工具选型与风险管理」模块；录屏发 B 站
5. **D5+**：给全部 50 名提交者发「你的工具栈风险评分」邮件（按 OpenAI 依赖度自动分级），附 AI PM Bootcamp「工具风险管理模块」试听链接；高依赖度企业额外触发 AI Adoption 企业培训咨询邀请
