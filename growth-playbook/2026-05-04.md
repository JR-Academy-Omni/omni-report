## 玩法 ① **母亲节 AI 礼物生成器 × Prompt Lab 引流**（内容钩子 + 节日营销）

|---|---|

| **对应业务** | Prompt Lab（试用激活） → 后端漏斗到 AI Adoption Bootcamp |
| **目标用户** | 拉新（泛华人职场人群）+ 激活（Prompt Lab 注册转化） |
| **执行难度** | 🟢 小（Prompt Lab 已有能力，只需加一个节日包装落地页 + 分享流） |
| **预估指标** | 5/4–5/11 共 7 天：1,200 次生成，280 次小红书/Instagram 分享，80 人注册 Prompt Lab，其中 12% 转化付费课（约 10 人） |

3. **D1–D2**：找 5 位 JR 学员（现成内部资源）先体验并发小红书，带话题 #AI帮我给妈妈写信，做话题冷启动
4. **D2–D7**：小红书投 3 篇 KOC 测评（内容：教程类"我用AI给妈妈写了封信，她哭了"），预算 $300 AUD；同时发微信朋友圈引导 JR 社区老学员转发
5. **D7（5/11 母亲节当天）**：统计生成量 + 分享量，Top 3 分享量发 Prompt Lab 年度会员；同日推送邮件："你已经体验过 Prompt Lab，下一步是 AI Adoption Bootcamp"

## 玩法 ② **TAFE/Microsoft 入门结业 → JR 实战接力券**（身份认证折扣 + 裂变）

| 字段 | 值 |
|---|---|
| **一句话描述** | 上传 TAFE 或 Microsoft AI Essentials 结业证书截图，自动领 $300 off AI Adoption Bootcamp 优惠码；把码转发给 1 个同样有证书的朋友，朋友也得 $200 off，自己再得 $100 credit |
| **灵感来源** | GitHub Student Pack（身份认证折扣）+ 瑞幸邀请各得一杯（双向裂变） |
| **对应业务** | AI Adoption Bootcamp（付费转化 B2C）→ 未来可延伸到企业团购 |
| **目标用户** | 付费转化（已有 AI 基础认知、正在找"下一步"的澳洲职场人） |
| **执行难度** | 🟢 小（落地页 + 证书图片上传 + 邮件优惠码发送，无需后端开发；用 Typeform + Zapier 可 1 天搭通） |
| **预估指标** | 3 周内：150 人提交证书验证，25% 转化付费报名 = 37 人；裂变二级触达 80 人，其中 15 人报名；合计 3 周新增 ~50 人报名，CAC < $100 AUD |
| **本周做？** | ✅ 推荐本周启动——Microsoft 300 万人培训（2026-04-23 宣布）余热正旺，TAFE 学员每月持续结业 |

**落地步骤**：
1. **D0**：写落地页文案（核心句："你学完了 AI 基础，我们教你做第一个真实 AI 项目"），用 Webflow/Notion 搭 1 页；表单：姓名 + 邮件 + 上传证书截图
2. **D1**：配置 Zapier：收到提交 → 发 $300 off 邮件 + 附上"转发给同路人"专属链接（带 referral UTM）；被转发者填表验证 → 双方各得对应 credit
3. **D2**：在 LinkedIn AU 发一篇文章（JR 教研负责人署名）："完成了 Microsoft/TAFE AI 入门之后，真正的工程实践是什么样的？" — 文末插接力券链接
4. **D3–D7**：定向在 TAFE NSW/VIC 官方 Facebook 群组 + LinkedIn 「Microsoft AI Skills」相关话题帖子下留评，不发广告，留真实有价值评论并附链接
5. **D10**：汇总报名数据；若转化率 >20%，下一步向 Microsoft AU Education 团队申请官方合作，把接力券变成官方推荐资源

## 玩法 ③ **Agent Governance Sprint：48 小时给你的 AI 应用加锁**（挑战赛 + 徽章）

| 字段 | 值 |
|---|---|
| **一句话描述** | 报名后领"AI Agent 治理挑战包"（7 道企业场景题），48 小时内提交解决方案截图 + 100 字说明，前 20 名完成者得 AI Engineer Bootcamp Enterprise Track 免费试听资格 + 可发 LinkedIn 的"Agent Governance Certified"数字徽章 |
| **灵感来源** | LinkedIn Skills Badge（徽章可见性驱动参与）+ Strava 月度挑战（有限时窗口的完成感） |
| **对应业务** | AI Engineer Bootcamp（高质量拉新 + 付费转化） |
| **目标用户** | 拉新（在职工程师，特别是已在企业用 AI Agent 的）+ 付费转化 |
| **执行难度** | 🟡 中（需要出 7 道有质量的题目 + 徽章设计 + 提交审核流程；预计 2 天准备） |
| **预估指标** | 1 周：350 人报名，40% 完成提交（140 人），其中 20% 预约 Enterprise Track 试听 = 28 人，15% 转化付费 = 5 人高客单价报名（课程 ~$3k+ AUD） |
| **本周做？** | ✅ 推荐本周启动——Microsoft Agent 365 于 2026-05-01 GA（3 天前），是最佳蹭热点窗口；2 周后话题热度下降 |

**落地步骤**：
1. **D0**：教研负责人出 7 道题（参考 Agent 365 GA 公告中的 3 个核心场景：访问控制策略配置 / 审计日志查询 / 多 Agent 权限隔离），每题给参考资源链接，不出纯理论题
2. **D1**：设计徽章（SVG，可上传 LinkedIn Credential）；搭提交表单（Google Form + 截图上传）；写报名落地页，标题"你的 AI Agent 在企业里安全吗？48 小时挑战"
3. **D1**：在 LinkedIn 发宣传文（JR 讲师署名），正文拆解 Agent 365 的访问控制机制，末尾接挑战赛报名链接；同步在 JR 学员社区内部发起，前 50 报名者额外得 JR 学员内推池优先资格
4. **D2–D3（48h 窗口）**：每天在 LinkedIn/小红书发"今日题目提示"短视频（2 分钟），保持话题热度，引导围观者报名
5. **D4**：公布前 20 名名单，给徽章；同日发邮件序列：完成者 → 推 Enterprise Track 试听；未完成者 → 推"AI Agent 治理基础课"免费视频引流

## 玩法 ④ **开源模型盲测闯关 × /learn SEO 激活**（答题瓜分 + streak）

| 字段 | 值 |
|---|---|
| **一句话描述** | 在 /learn 页面嵌入"本周模型大乱斗"互动题：给出 3 段匿名代码/文本输出，猜是 Gemma 4 / Kimi K2.6 / Mistral 128B / DeepSeek V4 / Muse Spark 中的哪个生成的；答对全部 3 题领 Vibe Coding 7 天免费试用；分享成绩截图到小红书 #JR模型大乱斗 额外抽 1 名 Bootcamp 全额奖学金 |
| **灵感来源** | 知乎答题瓜分奖金 + Wordle 每日一题（分享截图形成社交传播） |
| **对应业务** | AI 学习中心 /learn（SEO 流量激活）→ Vibe Coding Bootcamp（试用转付费） |
| **目标用户** | 激活（/learn 流量变注册）+ 拉新（小红书话题带来新流量） |
| **执行难度** | 🟡 中（需要出 3 道有辨识度的盲测题 + /learn 页面嵌入互动组件 + 小红书话题运营；前端 1 天，题目半天） |
| **预估指标** | 1 周：800 次答题参与，18% 全部答对并领试用 = 144 次 Vibe Coding 试用激活，小红书分享 120 条触达 4,000+，3% 转化付费 = 4 人；话题存活时间 2 周以上（模型热度持续） |
| **本周做？** | ✅ 推荐本周启动——Gemma 4 + Muse Spark + Mistral 128B 同周亮相，"选哪个模型"是本周最热搜索词；下周话题热度减半 |

**落地步骤**：
1. **D0**：教研团队选 3 个有代表性的任务（如：写一段 Python 爬虫 / 解释一个法律条款 / 生成一段营销文案），用 5 款模型各跑一次，保留最有辨识度的 3 个对比对（即答对率在 40-70% 之间，太简单没传播，太难没人做完）
2. **D1**：在 /learn 页面嵌入互动组件（可用 Typeform 嵌入，3 道题 + 计时器 + 结果页）；答对 3/3 自动触发 Vibe Coding 试用码邮件；结果页提供"分享你的成绩"按钮（预填写 Twitter/小红书文案）
3. **D1**：运营在小红书发话题引导帖"你能分辨 Gemma 4 和 DeepSeek V4 吗？来挑战"，附答题链接；同时在 JR 学员微信群发起"挑战群排行榜"
4. **D2–D5**：每天公布"全球答题正确率"（/learn 后台统计），用数据更新小红书动态，保持话题热度；答题人数破 300 发一次里程碑推送
5. **D7**：直播公布奖学金得主，同时发布"5 款模型的正确答案解析"文章（知乎/B站），这篇文章作为长尾内容持续引流 3 个月

## 玩法 ⑤ **AI 薪资差距测算器 × AI Engineer 转化**（内容钩子 + 求职裂变）

| 字段 | 值 |
|---|---|
| **一句话描述** | 3 分钟表单：输入当前岗位 / 年薪 / 已掌握技能；AI 自动生成"你离 SEEK $152k AI Engineer 均薪还差 X 个技能"个性化 PDF 报告；报告末尾匹配对应 Bootcamp 章节 + "预约 1v1 技能评估"CTA；分享报告截图可额外得 $100 Bootcamp credit |
| **灵感来源** | 测 AI 水平推课程（内容钩子）+ Notion 邀请赚 Credit（社交分享驱动） |
| **对应业务** | AI Engineer Bootcamp（付费转化）+ 求职服务 Job Referral（进内推池） |
| **目标用户** | 付费转化（在职程序员 / 想转型 AI Engineer 的职场人）+ 求职 Referral 新用户激活 |
| **执行难度** | 🟡 中（表单 + Claude API 生成个性化报告文本 + PDF 模板 + 邮件序列；约 3 天开发） |
| **预估指标** | 2 周：500 次完成测算，25% 预约 1v1 技能评估 = 125 次预约，20% 转化付费 Bootcamp = 25 人；分享截图 60 次，二级触达 2,000+；Job Referral 新增 40 人进内推池 |
| **本周做？** | ⏳ 下周启动（SEEK $152k 数据本周新鲜，但工具需 3 天开发；下周一上线正好赶上本周热度余温；本周可先出 MVP 用 Typeform 手动生成报告验证需求） |

1. **本周**：用 Typeform 搭 MVP 版（表单收集数据，人工在 24 小时内发 PDF 报告，验证用户是否真的预约 1v1）；如果 D3 完成率 >30%，启动自动化开发
2. **D0（下周一）**：上线自动化版：Typeform 提交 → Claude API 根据填写技能与 AI Engineer JD 关键词做 gap 分析 → 生成 Markdown 报告 → WeasyPrint 转 PDF → 自动邮件发送
3. **D1**：在 LinkedIn 发文："我测了 100 个工程师的 AI 技能差距，发现了 3 个最常见的盲区"（用 MVP 阶段的真实数据，保持匿名），文末接测算器链接；同步在 JR 学员社区推送，让学员帮分享
4. **D3**：统计完成率 + 预约率；给所有完成测算但未预约的人发"你的第一个技能缺口可以免费学，这里是资源"邮件（把 /learn 相关章节作为免费内容发过去，二次激活）
5. **D7**：整理数据做「澳洲工程师 AI 技能缺口图谱」发知乎/B站（样本越多越有说服力），作为长尾引流内容持续运营

## 这周不推的玩法

- **"拼团 5 人砍价 50%"** — JR Bootcamp 客单价 $2,000–$4,000 AUD，砍价模式在这个价格带里只会压缩利润、筛出价格最敏感的用户，而这批用户完课率和转介绍率都偏低。适合日用品，不适合高决策门槛的职业培训。
- **"May the 4th Be With You" 星球大战主题活动** — 今天是 5/4，确实是西方营销传统节日，但 JR 受众以澳洲华人职场人为主；星球大战 IP 联名需要版权授权，临时蹭话题只能用谐音梗，对技术向 Bootcamp 品牌调性帮助有限，效果不如用母亲节（5/11）做真实情感钩子。
