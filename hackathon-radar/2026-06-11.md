## 🧑‍💻 Solo Founder 案例雷达（2 条）

**案例 1：Mattia（Sleek.design） — AI 设计工具，6 周 $10K MRR，零付费推广**

Mattia 一个人花 3 周用 AI 工具搭出 Sleek：一款给不懂设计的 app 开发者「vibe design」移动端界面的工具。发布后靠清晰定位 + 垂直内容 SEO，6 周内跑到 $10K MRR，没花一分钱广告费。核心打法：不做通用 UI 工具，只做「手机 app 界面」，够窄才能够快。

- **来源：** [Indie Hackers](https://www.indiehackers.com/post/tech/hitting-10k-mrr-in-six-weeks-with-an-ai-design-tool-pEvmU5qkWS6ny0AR9SUv)
- **可做小红书/讲座选题角度：** 「一个工具，一个场景，6 周从 0 做到 $10K MRR——AI 时代的极简创业公式」

---

**案例 2：Samuel Rondot — StoryShort.ai，$24,881/月，前眼镜师自学编程做 AI SaaS 🆕**

Samuel 是法国人，没有 CS 学位，原职业是眼镜师。自学编程后做了 StoryShort.ai（AI 无脸视频生成器，一键把博客/脚本转成 TikTok/YouTube Shorts），加上第二款产品 Capacity.so，整体产品组合已达 $28K/月。核心策略：不追热点，每个产品只解决一个具体工作流，快速验证需求 → 最小 MVP → 获客 → 迭代。StoryShort 的月收入通过 Stripe API 在 TrustMRR 公开可查。

- **来源：** [Indie Hackers](https://www.indiehackers.com/post/tech/learning-to-code-and-building-a-28k-mo-portfolio-of-saas-products-OA5p18fXtvHGxP9xTAwG) · [TrustMRR](https://trustmrr.com/startup/storyshort)
- **可做小红书/讲座选题角度：** 「从磨眼镜到月入 $2.5 万：他用一个 AI 工具，只解决一件事」— OPC 创业营招生/内容最典型的「非技术人背景」案例

## 🎯 本期行动建议

### 给创业营学员：

**① Build with Gemini XPRIZE（最值得冲，截止 8/17）**
为什么是你：规则要求「真实用户 + 真实收入」，不是拼 demo 技术，是拼商业落地——这正是 OPC 创业营在训练的。第一步做什么：今晚去 [xprize.devpost.com](https://xprize.devpost.com/) 注册账号，把你现有产品/MVP 改接一个 Google Cloud 服务（Vertex AI 或 Cloud Run 都行），记录从今天开始的真实用户增长数据作为参赛证明。

**② Band of Agents Hackathon（本周五 6/19 截止，最快见效）**
为什么是你：$10K+ 奖金，7 天在线，deadline 近所以竞争者少。第一步做什么：如果你有任何现成的 Claude/GPT workflow 代码，明天就改成调用 Band 框架的多 agent 版本，用现有项目直接交一个 demo 就行，不需要从零开始。
