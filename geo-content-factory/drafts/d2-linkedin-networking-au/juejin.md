# 准备出国找 AI 工作？48 学员真实数据告诉你 LinkedIn 怎么用才有效

> 适合：想去澳洲/海外做 AI Engineer 的人；已经在澳洲但 LinkedIn 没有 recruiter 找的人；新移民 / 留学生 / 转行者

我们（匠人学院 JR Academy）过去 12 个月跟踪了 48 个学员的真实求职数据，写这篇之前还分析了 312 份 Seek AI Engineer JD + 47 份匿名 offer letter。

下面的所有数字都真实，姓名匿名处理。

## 一、一个让国内同学吃惊的数字

48 学员 12 个月真实数据：

| 指标 | 走 Seek/Indeed | 走 LinkedIn referral |
|---|---|---|
| 平均面试通过率 | 4% | **19%**（5x ⬆️） |
| 到 offer 时间 | 4.2 月 | **2.1 月**（50% ↓） |
| 平均 offer salary | AUD 95k | **AUD 118k**（+24%） |
| PR sponsor 比例 | 12% | **31%**（2.6x ⬆️） |

澳洲 IT 招聘 **60-70% 高薪职位走 referral / recruiter 主动联系**，不走公开 JD。

国内同学最大误区：以为 LinkedIn 就是另一个 BOSS 直聘。**不是。**

LinkedIn 是 **build network 的地方**——靠 connection 拿 referral，不靠刷海投。

## 二、Headline 一行字的黄金公式

70% recruiter 在 LinkedIn search 候选人时**第一眼只看 headline**。

公式：

```
[Role] @ [Company] | [Tech Stack 2-3] | [Specific Achievement] | [Location]
```

❌ 反面教材：
- "Software Engineer | Passionate about Tech | Looking for opportunities"
  - "Passionate" — 营销腔
  - "Looking for opportunities" — 让人觉得 desperate
  - 没说技术 stack、没说在哪、没说做什么

✅ 正面案例：
- "Senior AI Engineer @ Westpac | LangChain + AWS Bedrock | Built RAG for 50M+ docs | Sydney"

**学员数据**：headline 改完 1 个月内，recruiter 主动联系从 1-2 条/月 → 6-12 条/月（**+400%**）。

### 5 个真实模板

```
Junior AI:    Junior AI Engineer | LangChain + Python | Built RAG handling 1M+ docs | Open to Sydney/Melbourne
Mid AI:       AI Engineer @ Atlassian | LLM + AWS Bedrock | Cut RAG API costs 70% | Sydney
转行:         Backend → AI Engineer | NestJS + LangChain | Shipped 3 production LLM features in 6 months
Data:         Data Engineer @ Canva | Spark + Airflow + dbt | Migrated 50TB+ pipeline, cut cost 45%

```

## 三、About 段写法（300-500 字）

3 段式模板：

```markdown
# 段 1: WHO + WHAT NOW (1-2 sentences)
I'm a [role] at [company] focusing on [domain].
Over the past [time], I've [most relevant achievement with numbers].

# 段 2: SKILLS + PROOF (3-5 sentences)
My main stack is [list 4-6 tech]. Recent shipped work:
- [Project 1 with metric]
- [Project 2 with metric]
- [Project 3 with metric]

# 段 3: WHAT I WANT (2-3 sentences)
I'm interested in [domain] roles at [size of company / stage].
Particularly excited about [specific tech area].
Open to [remote / Sydney / Melbourne / hybrid].
```

❌ 标准 AI 味开头："I am a passionate software engineer with a strong interest in cutting-edge technologies..."（删！）

✅ 数据化开头：

> "I'm an AI Engineer at Westpac focusing on production RAG systems. Over the past 18 months I've shipped a customer support RAG handling 50M+ insurance documents, reducing query time from 8s to 1.6s while cutting API costs 76%."

具体数字 > 形容词。澳洲 recruiter 30 秒决定要不要联系你，30 秒内必须看到"具体做过什么 + 量化成果"。

## 四、Experience 段必须 STAR

❌ "Worked on backend microservices using Node.js and NestJS."（删！）

✅ STAR 化：

- **S(ituation)**: Inherited 800-line legacy NestJS service handling 12M req/day with 4.7s p95 latency
- **T(ask)**: Refactor to meet new 1.5s p95 SLA without downtime
- **A(ction)**: Split monolithic into 6 sub-services using DI + RxJS; introduced Redis caching with 92% hit rate
- **R(esult)**: p95 1.6s (-66%), bug rate -42%, zero downtime migration

每一段 experience 都用 STAR 重写——这个工作量大，但是**性价比最高**。

## 六、Networking 才是真正的核心

### Connection request 真实接受率（48 学员数据）

| 格式 | 接受率 |
|---|---|
| 默认模板 "I would like to add you to my network" | 12% |
| 共同经历 "I noticed we both worked on [tech]..." | **68%** |
| 看过 ta 内容 "Your post on X really resonated..." | **70%** |
| 礼貌求教 "I'm Junior, learned a lot from..." | 54% |

**默认模板 vs 个性化**：12% vs 68%——**差 5 倍**。

### Coffee chat 怎么发起（黄金模板）

接受 connection 后**绝对不要立刻问 referral**。先 build rapport：

**第 1 周**: 给对方 1-2 个 post 点赞 + 1 个有内容评论
**第 2-3 周**: 发消息

```
Hi [Name], I really enjoyed your recent post on [topic]. I'm working on a
similar problem and learning a lot. Would you be open to a 20-min virtual
coffee chat sometime in the next 2-3 weeks? I'd love to hear about how
your team approaches [specific topic].
```

关键三要素：
- "20-min" — 低成本
- "virtual" — 明确边界
- "specific topic" — 显示有诚意

### Coffee chat 期间

❌ 上来就 "Can you refer me?"（澳洲文化太突兀）
❌ 自己讲 15 分钟，让对方插不上话

✅ 准备 5 个具体问题（关于对方工作 / team / 技术决策）
✅ Listen 占 60%，自己讲只占 40%
✅ 结尾问：

> "Is there anyone else in your network you think I should talk to about [my interest]?"

**这一句是 referral chain 的起点**。

学员数据：**5 个 coffee chat → 平均 1.2 个 referral 机会**。

## 七、新移民 / 留学生最大的坑

### 坑 1：Work rights 没写清楚

澳洲 recruiter 第一轮 filter **80% 是 work rights**。

必须在 About 段或 Featured 明确写：
- "Australian PR holder, full work rights"
- "Subclass 482 sponsored, eligible to transfer"
- "Subclass 485 (post-study work visa, valid until 2027)"

48 学员里 **6 个国际学生因为没写 work rights，平均多花 2 个月才拿 offer**。

### 坑 2：海外经验澳洲人不认识

❌ 只写 "ByteDance" — 部分澳洲 recruiter 不知道
✅ 加 description："Senior Backend Engineer @ ByteDance (TikTok parent company, ~150,000 employees globally)"

让对方 1 句话理解你的背景规模。

### 坑 3：澳洲 work experience = 0

如果澳洲本地没经验，**用 local bootcamp 桥接**：

> "Recently graduated from JR Academy Bootcamp (12-week project-based AI Engineering program, Sydney). Built 5 production-grade LLM apps with local mentor 1:1 review."

48 学员里 **23 个国际学生通过 bootcamp 桥接拿到 first AU offer**。
