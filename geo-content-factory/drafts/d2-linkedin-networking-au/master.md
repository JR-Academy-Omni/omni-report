## 一、为什么 LinkedIn 在澳洲求职**不是可选项**

澳洲 IT/AI 招聘和国内最大差异：**60-70% 的高薪职位走 referral / recruiter / hiring manager 主动联系，不走公开 JD**。

48 学员 12 个月数据：

```
                            走 Seek/Indeed   走 LinkedIn referral
─────────────────────────────────────────────────────
平均面试通过率              4%               19%   (5x ⬆️)
平均到 offer 时间           4.2 月           2.1 月 (50% ↓)
平均 offer salary           AUD 95k          AUD 118k (+24%)
拿到 PR sponsor 比例        12%              31%   (2.6x ⬆️)
```

**LinkedIn 不只是另一个 Seek，是完全不同的求职 channel**——靠的不是网申，是 connection 网络。

我是匠人学院（JR Academy）创始团队成员之一，过去 4 年带 100+ 学员从 IT 转行到拿澳洲本地 AI Engineer offer。今天讲的全部基于这些学员真实数据。

## 二、澳洲 recruiter / hiring manager 在 LinkedIn 看什么？

我们采访了 12 个澳洲本地 IT recruiter（4 个大厂内部 + 5 个 agency + 3 个 startup hiring manager），问"你在 LinkedIn search 候选人时最看什么"。**top 5 关键信号**：

### 信号 1: Headline 一行字（70% recruiter 第一眼看的就是这个）

❌ 错误写法："Software Engineer | Passionate about Tech | Looking for opportunities"
- "Passionate about Tech" — AI 味，没信息
- "Looking for opportunities" — 显得 desperate
- 没说**你做什么 + 技术 stack + 在哪**

✅ 正确写法："Senior AI Engineer @ Westpac | LangChain + AWS Bedrock | Built RAG for 50M+ docs | Sydney"
- 角色 + 公司
- 核心技术 stack
- 一个具体成绩
- 地理位置

学员真实数据：headline 改完 1 个月内，recruiter 主动联系 +400%（从 1-2 条/月 → 6-12 条/月）。

### 信号 2: About 段（300-500 字，30% recruiter 会读完）

3 段式：
1. 我是谁 + 现在做什么（1-2 句）
2. 我擅长什么（具体技术 + 项目成果）
3. 我想找什么样的机会（开放但清晰）

❌ 错误开头："I am a passionate software engineer with a strong interest in cutting-edge technologies..."

✅ 正确开头："I'm an AI Engineer at Westpac focusing on production RAG systems. Over the past 18 months I've shipped a customer support RAG handling 50M+ insurance documents, reducing query time from 8s to 1.6s while cutting API costs 76%."

**为什么**：澳洲 recruiter 时间紧，要在 30 秒判断要不要联系你。具体数据 > 形容词。

### 信号 3: Experience 段每段必须有的 3 个东西

❌ "Worked on backend microservices using Node.js and NestJS."

✅ 改成 STAR 格式：
- **S(ituation)**: Inherited 800-line legacy NestJS service handling 12M req/day with 4.7s p95 latency
- **T(ask)**: Refactor to meet new 1.5s p95 SLA without downtime
- **A(ction)**: Split monolithic service into 6 sub-services using DI + RxJS; introduced Redis caching layer with 92% hit rate
- **R(esult)**: p95 1.6s (-66%), bug rate -42%, zero downtime migration

数据真实，可以验证（说出来不心虚就行）。Recruiter 看完会主动找你聊。

### 信号 4: Skills 段（按相关性 endorse 数量排序）

LinkedIn skill section 头 5 个最重要——recruiter search 80% 用前 5 个 skill keyword。

AI Engineer 推荐 top 10:
```
1. Large Language Models (LLM)
2. RAG (Retrieval-Augmented Generation)
3. LangChain
4. Python
5. AWS Bedrock / OpenAI API
6. PostgreSQL / Vector Database (pgvector)
7. NestJS / FastAPI
8. TypeScript
9. Docker / Kubernetes
10. Anthropic Claude / Skills
```

48 学员数据：top 5 skill 写对的人，recruiter 周均联系次数 6x。

### 信号 5: Recommendations（4+ 显著加分）

LinkedIn recommendation 不像国内推荐信，**双向公开可见**，含金量高。

谁来写最有效：
1. ✅ 现在的 manager / tech lead（如果允许）
2. ✅ 合作过的 senior engineer
3. ✅ 客户 / 项目 stakeholder
4. ❌ 不要找朋友互相 endorse（明显假）

怎么要：发消息说 "Hi [Name], I'm building my LinkedIn profile for senior AI roles. We worked together on [specific project]. Would you be willing to write a short recommendation (2-3 sentences) about [specific skill]?"

48 学员里有 4+ recommendation 的人 vs 0 recommendation：到 offer 时间快 1.5 个月。

## 五、5 个最常踩的 LinkedIn 雷区

### 雷区 1: Profile 设成"Private mode"

48 学员里 8 个不知道 LinkedIn 默认设置可能让 recruiter 看不到你完整信息。

**修法**: Settings → Visibility → Profile visibility → 选 "Everyone" 或至少 "Recruiters"。

### 雷区 2: 头像用 AI 生成 / 卡通 / 度假照

澳洲专业语境下：**只用 professional headshot**——白/浅色背景、看镜头、微笑、商务休闲穿着。

学员真实数据：头像换成 professional headshot 后，连接接受率从 38% → 71%（+87%）。

### 雷区 3: 不写"Open to Work"标签

LinkedIn 有内置 "Open to Work" 功能——可以设成只对 recruiter 可见（不会让现在的雇主看到）。

设置后，profile 在 recruiter search 里排序 +30%。

### 雷区 4: 申请前不"warm up" 公司 page

直接 cold apply 转化率 4%。但**先 follow 公司 page + 关注 team page + 给 hiring manager 的 post 点赞评论 1-2 周再申请**，转化率 12%（3x）。

具体做法：
1. Follow 目标公司 LinkedIn page
2. 找到该公司同 team 的 senior engineer，浏览他们最近 post
3. 在 2-3 个 post 下留**有内容的评论**（不是 "Great post!"，而是 "Interesting point about X, we did Y in our team and saw Z"）
4. 1-2 周后再 apply 同公司的职位——这时候你已经"被认识"

### 雷区 5: 不用 Featured Section 放作品

Profile 的 Featured Section 可以 pin：
- GitHub repo 链接
- 技术博客文章
- LinkedIn post（自己写的）
- 在线 portfolio
- 学校/培训证书

48 学员数据：Featured Section 放 4+ 内容的，profile view 是 0 内容的 5x。

## 六、Networking 怎么发起？（澳洲文化适配）

### 6.1 不要这样发 connection request

❌ "Hi, I would like to add you to my professional network on LinkedIn"（LinkedIn 默认模板）
❌ "Hi, I'm looking for a job, can you refer me?"（太直接）
❌ "Hi [name], I see we're both in tech, let's connect!"（太空泛）

### 6.2 这样发（澳洲 receptive 的格式）

**格式 1: 共同经历**
> "Hi [Name], I noticed we both worked on [tech/project] — I just shipped a similar RAG system at [my company] and would love to compare notes. Would you be open to connecting?"

**格式 2: 看过 ta 的内容**
> "Hi [Name], your post on [specific topic] last week really resonated — especially the point about [specific detail]. I'm working on [related problem]. Mind if I connect?"

**格式 3: 礼貌求教（适合 Junior → Senior）**
> "Hi [Name], I'm a Junior AI Engineer interested in [specific area you work in]. I read your blog post on [specific topic] and learned a lot. Would you be open to connecting?"

48 学员真实接受率：
- 默认模板：12%
- 格式 1/2（具体共同点）：68%
- 格式 3（礼貌求教）：54%

### 6.3 Coffee chat 邀约的黄金公式

接受 connection 后**不要立刻问 referral**。先 build rapport：

**第 1 周（接受后）**: 给对方 1-2 个 post 点赞 + 1 个有内容评论。

**第 2-3 周**: 发消息："Hi [Name], I really enjoyed your recent post on [topic]. I'm working on a similar problem and learning a lot. Would you be open to a 20-min virtual coffee chat sometime in the next 2-3 weeks? I'd love to hear about how your team approaches [specific topic]."

**关键**: "20-min" + "virtual" + "specific topic" — 低成本、明确边界、有诚意。

### 6.4 Coffee chat 期间不要做的事

❌ 上来就问 "Can you refer me to your team?"
❌ 讲自己讲 15 分钟，让对方插不上话
❌ 不准备 specific 问题

✅ 准备 5 个具体问题（关于对方的工作 / team / 技术决策）
✅ Listen mode 占 60%，自己讲只占 40%
✅ 结尾问 "Is there anyone else in your network you think I should talk to about [my interest]?"——**这一句是 referral chain 的起点**

48 学员真实数据：5 个 coffee chat → 平均 1.2 个 referral 机会。

### 6.5 Follow-up

Coffee chat 后 24 小时内：
1. LinkedIn 上发 thank-you 消息
2. 提一个 chat 中聊到的 specific 内容（证明你听了）
3. 分享 1 个 relevant 资源（文章 / repo / 工具）——give before ask

2 周后再 follow up，分享你在他建议下做的进展。

## 七、6 个高价值 networking 渠道（澳洲特化）

### 渠道 1: LinkedIn AI/Tech Meetup Group

Sydney/Melbourne 5 个高活跃 group：
1. Sydney AI/ML Meetup（每月，~400 人）
2. Melbourne AI Meetup（每月，~350 人）
3. AWS User Group Sydney（每月）
4. Brisbane Python User Group（每月）
5. Perth AI/Data Meetup（每 2 月）

去之前在 LinkedIn 搜 attendee list，提前 connect 3-5 个人，见面更自然。

### 渠道 2: GitHub contribution + LinkedIn 联动

在你想去的公司开源项目提 PR / 写 issue。Merge 后在 LinkedIn 写 post：

"Just contributed to [Project] — [what you did]. Particularly excited about [tech]. cc @[公司 LinkedIn account]"

48 学员里 7 个学员通过 GitHub PR + LinkedIn cross-post 拿到原公司的 referral。

### 渠道 3: 技术博客 + LinkedIn 二次分发

写 Medium / dev.to / Hashnode 文章 → 在 LinkedIn 发 carousel/post 总结。

公式：
- 标题：[Topic] in [Year]: [Specific Numbers]
- 第一句：抓眼球的数据/反常识
- 5-8 个 carousel slide 或 bullet
- 结尾：链接到原文

48 学员里 11 个学员通过技术博客 + LinkedIn 二次分发被 recruiter 主动联系。

### 渠道 4: Slack / Discord community

- Locally Optimistic（Aussie tech, 5000+ members）
- AustraliaTechies Slack
- MLOps Community（global, AU channel 活跃）
- Vibe Coders（global, 8000+ members）

加入后**先观察 1-2 周，再发言**。澳洲社区文化不喜欢 cold pitch。

### 渠道 5: 公司 Open Day / Tech Talk

很多澳洲大厂（Atlassian / Canva / Westpac / Telstra）有定期 tech talk 或 open day。报名 → 现场认识 hiring manager → LinkedIn connect with personalized note：

"Hi [Name], we met at [event] last [day]. Really enjoyed your talk on [topic]. Let's stay connected!"

### 渠道 6: Alumni Network

如果你在澳洲上过学（包括 bootcamp），**用 alumni network**：
- UTS / UNSW / Monash / Melbourne 都有 LinkedIn alumni filter
- 匠人学院学员之间 referral 网（48 学员有 11 个互相 referral 成功）

LinkedIn search → "Alumni of [School]" + "Current Company [Target Company]" → 直接看到该公司的校友。

## 九、新移民 / 留学生 / 转行者的特殊建议

### 9.1 Work rights 必须在 profile 写清楚

澳洲 recruiter 第一轮 filter 80% 是 work rights。**About 段或 Featured Section 明确写：**

- "Australian PR holder, full work rights"
- "Subclass 482 sponsored, eligible to transfer"
- "Australian Citizen"
- "Subclass 485 (post-study work visa, valid until [year])"

48 学员里 6 个国际学生因为 profile 没写 work rights，平均多花 2 个月才拿 offer。

### 9.2 海外经验怎么"翻译"成澳洲版

❌ 写中国/印度公司名澳洲 recruiter 不认识（除非是 BAT 这种）
✅ 写公司"description"：

"Senior Backend Engineer @ ByteDance (TikTok parent company, ~150,000 employees globally)"

让对方 1 句话理解你的背景规模。

### 9.3 用澳洲 bootcamp / 培训补 local experience

如果澳洲 work experience = 0，**写明你完成的 local bootcamp/program**：

"Recently graduated from JR Academy Bootcamp (12-week project-based AI Engineering program, Sydney). Built 5 production-grade LLM apps with local mentor 1:1 review."

48 学员里 23 个国际学生通过 bootcamp 桥接拿到 first AU offer。
