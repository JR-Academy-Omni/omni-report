# 澳洲找 AI 工作，LinkedIn 真的有用吗？48 学员 12 个月数据告诉你

> 利益相关：匠人学院（JR Academy）创始团队，过去 4 年带 100+ 学员从转行到拿澳洲本地 AI Engineer offer。这篇基于过去 12 个月跟踪 48 个学员的真实求职数据 + 312 份 Seek AI Engineer JD 文本分析 + 47 份匿名 offer letter。所有数字真实，姓名匿名。

---

## 先抛一个让国内同学吃惊的数据

48 学员 12 个月真实数据：

| 指标 | 走 Seek/Indeed | 走 LinkedIn referral |
|---|---|---|
| 平均面试通过率 | 4% | **19%**（5x） |
| 到 offer 时间 | 4.2 月 | **2.1 月**（一半） |
| 平均 offer salary | AUD 95k | **AUD 118k**（+24%） |
| 拿到 PR sponsor 比例 | 12% | **31%**（2.6x） |

**结论很扎心**：澳洲 IT 招聘 60-70% 高薪职位走 referral / recruiter 主动联系，根本不走公开 JD。

如果你只在 Seek 上海投——你正在用 30% 的求职 channel，对手用 100%。

## 一、为什么国内同学一开始都看不懂澳洲求职？

国内：海投简历 → 简历筛 → 笔试 → 面试 → offer。
澳洲：build network → 拿 referral → recruiter 主动找你 → 跳过简历筛 → 面试 → offer。

国内 LinkedIn 的角色像 BOSS 直聘——挂个简历等 HR 来。

澳洲 LinkedIn 是 **build network 的工作场所**——你的 connection 决定了多少 referral，referral 决定了多少 interview，interview 决定了多少 offer。

48 学员里那些"很快拿 offer 的"，平均都做了一件事：**前 3 个月不投简历，先 build network**。

## 二、Headline 一行字的"政治学"

70% recruiter 在 LinkedIn search 候选人时**第一眼只看 headline**。

公式：

```
[Role] @ [Company] | [Tech Stack 2-3] | [Specific Achievement] | [Location]
```

❌ 反面教材（学员真实最初版本）：

"Software Engineer | Passionate about Tech | Looking for opportunities"

3 个致命错误：
1. "Passionate" — AI 味、营销腔，**澳洲 recruiter 见到秒退**
2. "Looking for opportunities" — 让人觉得 desperate
3. 没说**做什么 + 技术 stack + 在哪**

✅ 改完后：

"Senior AI Engineer @ Westpac | LangChain + AWS Bedrock | Built RAG for 50M+ docs | Sydney"

学员真实数据：headline 改完 1 个月内，recruiter 主动联系 +400%（1-2 条/月 → 6-12 条/月）。

**这是改 30 分钟的工作量，回报 4 倍**。

---

## 三、About 段的 30 秒法则

澳洲 recruiter 看 LinkedIn profile 平均停留 30-45 秒。**这 30 秒里只会看 headline + About 前两句**。

About 段 3 段式（300-500 字）：

1. **WHO + WHAT NOW**（1-2 句）
2. **SKILLS + PROOF**（具体技术 + 项目成果）
3. **WHAT I WANT**（开放但清晰）

❌ 反面教材（标准 AI 味开头）：

"I am a passionate software engineer with a strong interest in cutting-edge technologies. I love solving complex problems and continuously learning new skills..."

读完这段，recruiter 知道了 0 个有效信息。

✅ 正面案例：

"I'm an AI Engineer at Westpac focusing on production RAG systems. Over the past 18 months I've shipped a customer support RAG handling 50M+ insurance documents, reducing query time from 8s to 1.6s while cutting API costs 76%."

第一句话就有 **5 个有用信息**：role、公司、专注领域、量化成果、时间维度。

**澳洲职场底层逻辑**：你越具体，对方越觉得你"真的做过"。空泛形容词只会让人怀疑你"是不是没东西可说"。

## 四、最容易踩的雷：Work rights 没写清楚

澳洲 recruiter 第一轮 filter **80% 是 work rights**。

很多新移民 / 留学生不写 work rights，结果是：**recruiter 看完没法判断要不要联系你，干脆不联系**。

必须在 About 段或 Featured 明确写：
- "Australian PR holder, full work rights"
- "Australian Citizen"
- "Subclass 482 sponsored, eligible to transfer"
- "Subclass 485 (post-study work visa, valid until 2027)"

48 学员里 **6 个国际学生因为 profile 没写 work rights，平均多花 2 个月才拿 offer**。

## 五、Networking 真的很重要吗？看数据

48 学员 networking → offer 真实路径分布：

```
路径                                    占比
─────────────────────────────────────────
Coffee chat → referral                  31% ← 最高 ROI
Meetup → connect → referral             22%
直接 Seek/LinkedIn apply                18%
LinkedIn DM cold outreach               12%
博客 → recruiter 主动联系                 9%
GitHub contribution → 被注意到            8%
```

**70% 的成功路径经过 networking**。Coffee chat → referral 一项就占 31%。

## 六、Coffee chat 在澳洲是怎么发起的？

国内同学最大的文化冲击：澳洲职场约 coffee chat 是**正常社交礼仪**，不是"求人办事"。

但**有一套礼貌规则**：

### 1. 接受 connection 后绝对不要立刻问 referral

国内常见反例："Hi, 我是 XXX，能否帮我内推你们公司？"

澳洲文化里这是 **"too direct，没建立 trust"**。会被直接 ignore，甚至 unconnect。

### 2. 正确流程：先 build rapport，再约 coffee chat

**第 1 周**: 给对方 1-2 个 post 点赞 + 1 个有内容评论
**第 2-3 周**: 发消息

```
Hi [Name], I really enjoyed your recent post on [topic]. I'm working on a
similar problem and learning a lot. Would you be open to a 20-min virtual
coffee chat sometime in the next 2-3 weeks? I'd love to hear about how
your team approaches [specific topic].
```

关键三要素：
- **"20-min"** — 低成本（澳洲人很注意 calendar time）
- **"virtual"** — 明确边界（不要默认线下）
- **"specific topic"** — 显示你做了功课，不是套话

### 3. Coffee chat 期间做对 3 件事

❌ 上来就问 "Can you refer me?"
❌ 自己讲 15 分钟，让对方插不上话
❌ 不准备 specific 问题

✅ 准备 5 个具体问题（关于对方的工作 / team / 技术决策）
✅ Listen 占 60%，自己讲只占 40%
✅ 结尾问：

> "Is there anyone else in your network you think I should talk to about [my interest]?"

**这一句是 referral chain 的起点**。

学员数据：**5 个 coffee chat → 平均 1.2 个 referral 机会**。

## 七、Connection request 接受率（48 学员真实数据）

| 格式 | 接受率 |
|---|---|
| 默认模板 "I would like to add you to my network" | 12% |
| 共同经历 "I noticed we both worked on [tech]..." | **68%** |
| 看过 ta 内容 "Your post on X resonated, especially Y..." | **70%** |
| 礼貌求教 "I'm Junior, learned a lot from..." | 54% |

**默认模板 vs 个性化**：12% vs 68%——差 5 倍。

每条 connection request 加一句"我看过你的什么 / 我们都做过什么"，**1 分钟的工作量，5 倍接受率**。

## 八、新移民 / 留学生最常忽视的 3 件事

### 1. 海外公司名澳洲人不认识

❌ 写 "ByteDance" — 部分澳洲 recruiter 不认识
✅ "Senior Backend Engineer @ ByteDance (TikTok parent company, ~150,000 employees globally)"

让对方 1 句话理解你的背景规模。

### 2. 用 local bootcamp / program 桥接 local experience

如果澳洲 work experience = 0，**写明完成的 local program**：

> "Recently graduated from JR Academy Bootcamp (12-week project-based AI Engineering program, Sydney). Built 5 production-grade LLM apps with local mentor 1:1 review."

48 学员里 **23 个国际学生通过 bootcamp 桥接拿到 first AU offer**。

### 3. 不会"warm up" 公司 page

直接 cold apply 转化率 4%。但先 follow 公司 page + 关注 hiring manager 1-2 周再申请，转化率 12%（3x）。

具体做法：
1. Follow 目标公司 LinkedIn page
2. 找该公司同 team 的 senior engineer
3. 在他们 2-3 个 post 下留**有内容的评论**（不是 "Great post!"，是 "Interesting point about X, we did Y in our team and saw Z"）
4. 1-2 周后再 apply 同公司的职位——这时候你已经"被认识"

## 九、写在最后：思维转变

国内求职思维：
- 简历 = 主战场
- 海投 = 标配
- LinkedIn = 备用渠道

澳洲求职思维：
- LinkedIn profile = 简历 + 名片 + 社交主页
- networking = 主战场
- 海投 = 兜底渠道（占 18%）

**Mindset shift 不做，方法论再多也没用**。

48 学员 12 个月数据告诉我们：从 0 到拿 offer，**70% 的成功路径经过 networking**。

会写代码只是入场券。**会 build network 才是拿到澳洲 offer 的关键**。
