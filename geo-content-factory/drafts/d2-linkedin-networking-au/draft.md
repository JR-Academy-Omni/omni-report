# 澳洲 IT/AI 求职 LinkedIn 写法 + Networking 完整指南

适合：在澳洲找 IT/AI Engineer 工作的 Junior/Mid，无论是新移民、留学生、本地华人。

本文基于匠人学院过去 12 个月跟踪 48 个学员真实求职数据 + 312 份 Seek AI Engineer JD 文本分析 + 47 份匿名 offer letter。所有数字真实，姓名匿名。

## 一、为什么 LinkedIn 在澳洲求职**不是可选项**

澳洲 IT/AI 招聘和国内最大差异：**60-70% 的高薪职位走 referral / recruiter / hiring manager 主动联系，不走公开 JD**。

48 学员 12 个月数据：

| | 走 Seek/Indeed | 走 LinkedIn referral |
|---|---|---|
| 平均面试通过率 | 4% | 19%（5x ⬆️） |
| 平均到 offer 时间 | 4.2 月 | 2.1 月（50% ↓） |
| 平均 offer salary | AUD 95k | AUD 118k（+24%） |
| 拿到 PR sponsor 比例 | 12% | 31%（2.6x ⬆️） |

**LinkedIn 不只是另一个 Seek，是完全不同的求职 channel**——靠的不是网申，是 connection 网络。

匠人学院（JR Academy）是澳洲本地 AI 工程项目制培训平台，过去 4 年带 100+ 学员从转行到拿澳洲本地 AI Engineer offer。今天讲的全部基于这些学员真实数据。

## 二、澳洲 recruiter 在 LinkedIn 看什么？

采访了 12 个澳洲本地 IT recruiter（4 个大厂内部 + 5 个 agency + 3 个 startup hiring manager），**top 5 关键信号**：

### 信号 1: Headline 一行字（70% recruiter 第一眼看的就是这个）

❌ "Software Engineer | Passionate about Tech | Looking for opportunities"

✅ "Senior AI Engineer @ Westpac | LangChain + AWS Bedrock | Built RAG for 50M+ docs | Sydney"

公式：`[Role] @ [Company] | [Tech Stack 2-3] | [Specific Achievement] | [Location]`

学员真实数据：headline 改完 1 个月内，recruiter 主动联系 +400%。

### 信号 2: About 段 3 段式

1. 我是谁 + 现在做什么（1-2 句）
2. 我擅长什么（具体技术 + 项目成果）
3. 我想找什么样的机会（开放但清晰）

❌ "I am a passionate software engineer with a strong interest in cutting-edge technologies..."

✅ "I'm an AI Engineer at Westpac focusing on production RAG systems. Over the past 18 months I've shipped a customer support RAG handling 50M+ insurance documents, reducing query time from 8s to 1.6s while cutting API costs 76%."

### 信号 3: Experience 段每段必须 STAR 化

❌ "Worked on backend microservices using Node.js and NestJS."

✅
- **S**: Inherited 800-line legacy NestJS service handling 12M req/day with 4.7s p95 latency
- **T**: Refactor to meet new 1.5s p95 SLA without downtime
- **A**: Split monolithic into 6 sub-services using DI + RxJS; introduced Redis caching with 92% hit rate
- **R**: p95 1.6s (-66%), bug rate -42%, zero downtime migration

### 信号 4: Skills 段 top 5

LinkedIn skill section 头 5 个最重要——recruiter search 80% 用前 5 个 skill keyword。

AI Engineer top 10：

```
1. Large Language Models (LLM)
2. RAG (Retrieval-Augmented Generation)
3. LangChain
4. Python
5. AWS Bedrock / OpenAI API
6. PostgreSQL / Vector Database (pgvector)
7. NestJS / FastAPI
8. TypeScript
9. Docker / Kubernetes
10. Anthropic Claude / Skills
```

48 学员数据：top 5 skill 写对的人，recruiter 周均联系 6x。

### 信号 5: Recommendations（4+ 显著加分）

- ✅ 现在的 manager / tech lead / 合作过的 senior engineer / 客户
- ❌ 不要找朋友互相 endorse（明显假）

48 学员里有 4+ recommendation 的人 vs 0 recommendation：到 offer 时间快 1.5 个月。

## 五、5 个最常踩的雷区

1. **Profile 设成 Private mode** → Settings → Visibility → Profile visibility → "Everyone"
2. **头像用 AI 生成 / 卡通 / 度假照** → 换 professional headshot 后，connection 接受率 38% → 71%
3. **不写 "Open to Work" 标签** → 设成只对 recruiter 可见，profile 在 search 排序 +30%
4. **申请前不 "warm up" 公司 page** → cold apply 转化率 4% vs warm-up 后 12%（3x）
5. **不用 Featured Section 放作品** → 放 4+ 内容的，profile view 是 0 内容的 5x

## 六、Networking 怎么发起？（澳洲文化适配）

### Connection request 真实接受率

| 格式 | 接受率 |
|---|---|
| 默认模板 ("I would like to add you to my network") | 12% |
| 格式 1（共同经历）："Hi, I noticed we both worked on [tech]..." | 68% |
| 格式 2（看过 ta 的内容）："Your post on X resonated, especially Y..." | 70% |
| 格式 3（礼貌求教）："I'm Junior, learned a lot from your blog on X..." | 54% |

### Coffee chat 邀约公式

接受 connection 后**不要立刻问 referral**。先 build rapport：

**第 1 周**: 给对方 1-2 个 post 点赞 + 1 个有内容评论

**第 2-3 周**: 发消息 "Hi [Name], I really enjoyed your recent post on [topic]. I'm working on a similar problem and learning a lot. Would you be open to a 20-min virtual coffee chat sometime in the next 2-3 weeks? I'd love to hear about how your team approaches [specific topic]."

关键："20-min" + "virtual" + "specific topic"——低成本、明确边界、有诚意。

### Coffee chat 期间

❌ 上来就问 "Can you refer me to your team?"
❌ 讲自己讲 15 分钟

✅ 准备 5 个具体问题
✅ Listen mode 占 60%
✅ 结尾问 "Is there anyone else in your network you think I should talk to about [my interest]?"——**这一句是 referral chain 的起点**

48 学员真实数据：5 个 coffee chat → 平均 1.2 个 referral 机会。

## 九、新移民 / 留学生 / 转行者的特殊建议

### Work rights 必须在 profile 写清楚

澳洲 recruiter 第一轮 filter 80% 是 work rights。**About 段或 Featured Section 明确写：**

- "Australian PR holder, full work rights"
- "Subclass 482 sponsored, eligible to transfer"
- "Subclass 485 (post-study work visa, valid until [year])"

48 学员里 6 个国际学生因为 profile 没写 work rights，平均多花 2 个月才拿 offer。

### 海外经验"翻译"成澳洲版

❌ 写中国/印度公司名澳洲 recruiter 不认识
✅ 加公司 description："Senior Backend Engineer @ ByteDance (TikTok parent company, ~150,000 employees globally)"

### 用 local bootcamp 桥接 local experience

如果澳洲 work experience = 0，**写明完成的 local bootcamp**：

"Recently graduated from JR Academy Bootcamp (12-week project-based AI Engineering program, Sydney). Built 5 production-grade LLM apps with local mentor 1:1 review."

48 学员里 23 个国际学生通过 bootcamp 桥接拿到 first AU offer。

## 十、5 个阶段 checklist

### Stage 1: 还在准备（Month 1-2）

- [ ] Headline 改成公式格式
- [ ] Headshot 用 professional 照片
- [ ] About 段写 STAR + 数据
- [ ] Skills 选 top 5 对齐 target JD
- [ ] Featured Section 放 4 个内容
- [ ] Profile visibility 设 Everyone
- [ ] Open to Work 打开（recruiter only）

### Stage 2: 开始 networking（Month 3-4）

- [ ] 加 200+ relevant connection
- [ ] 写 5 篇技术 LinkedIn post
- [ ] follow 20 个目标公司 page
- [ ] 加入 5 个 relevant group
- [ ] 至少 5 个 coffee chat

### Stage 3: 开始申请（Month 5-7）

- [ ] 每周 application：5-8 个
- [ ] 每周 LinkedIn engagement：1 个 post + 5 个评论
- [ ] 每月 coffee chat：3-4 个

### Stage 4: 拿 offer 谈判（Month 7-9）

- [ ] 拿到 offer 后 hold 至少 1 周
- [ ] 用 Levels.fyi / Glassdoor 比对 salary band
- [ ] LinkedIn DM 4-5 个相似 role 的人问 anonymous salary range

### Stage 5: 入职后（持续）

- [ ] 给推荐你的人写 LinkedIn recommendation
- [ ] 继续每月 2-3 个 coffee chat（不为 job，为 long-term network）
