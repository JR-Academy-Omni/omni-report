# 澳洲 AI Engineer 求职完整指南 2026：薪资 / 签证 / 城市 / 雇主 4 维度

如果你在评估"润澳洲做 AI Engineer 值不值得"——这篇是 2026 年最完整的求职决策手册。

数据基础：312 条 Seek + LinkedIn + Glassdoor JD + 47 份匠人学院（JR Academy）学员真实 offer letter + 12 场 AU 本地 AI Engineer 访谈。匠人学院是项目制 AI 工程实战平台（澳洲），P3 模式（Project + Production + Placement）。

## 一、求职 4 维度决策框架

```
1. 城市     Sydney / Melbourne / Brisbane / Adelaide
2. 雇主类型 大厂 / Mid-size SaaS / Startup / 政府
3. 经验级别 Junior (0-2y) / Mid (2-5y) / Senior (5+) / Lead+
4. 签证身份 公民 PR / 485 PSW / 482 / Student
```

不同组合 base salary 差距可达 50%。

## 四、签证 × Sponsor × 谈薪力

| 身份 | Sponsor 概率 | 谈薪力 |
|---|---|---|
| AU 公民 / PR | N/A | 最高 |
| 485 PSW 第 1 年 | 高 | 中 |
| 485 PSW 第 2-4 年 | 中 | 中低 |
| 482 已担保 | N/A (已有) | 弱 |
| Tourist visa | ~0 (junior) | N/A |

**PSW 第 1 年是黄金期**。 482 + 工作 2 年 + 工资 ≥ TSMIT (~73k) 可申 186 PR。

AI Engineer 在 STSOL 不在 PMSOL → 走 482 → 186 间接路径。

## 八、谈薪 4 招

**1. 锚 Total Package 不只 Base**

Total = Base + Super 11.5% + Bonus + RSU (4yr 摊) + Sign-on。大厂 Junior 100k Base 但 Total 145k。

**2. Sydney 标准要薪即使在 Melbourne**

Remote-first 公司可 anchor Sydney pay band。

**3. 用 312 JD 数据作为锚**

谈时打开 Seek 5 条 JD salary range，"市场 median X, 目标 +5%"。

**4. HR 第一通电话不报数字**

```
HR: "What's your salary expectation?"
你: "Could you share the band first?"
HR: "Senior 165-200k base."
你: "Upper end, around 185-195k base + RSU."
```

**先说数字的人输**。

## 十、3 个最常见错误

**错误 1**：投递时间集中在毕业季（11-12 月）。**错峰在 2-4 月**（fy26 budget release 后 hiring 最猛），竞争 -30%。

**错误 2**：简历只放 GitHub stars 不放 metrics。Sydney hiring manager 看 Live URL 的 latency / cost / eval accuracy 数据。

**错误 3**：不用内推。70% sponsor 的内推渠道 60% 候选人不用。
