## 三、4 类雇主对比

### 大厂 / 上市（Atlassian / Canva / CBA / Macquarie / Westpac / Telstra）

| 级别 | Base | Total |

| Lead+ | 230-300k | **300-450k** |

**特点**：流程 4-8 周；签证支持稳定（482 → 186 PR 通道清晰）；RSU 4 年 vest。

| 级别 | Base | Total |
|---|---|---|
| Junior | 90-105k | 100-115k |
| Senior | 160-200k | 200-260k |

**特点**：流程 2-4 周；equity 比例高 0.1-0.8%；签证可谈但要谈。

| 级别 | Base | Equity 4 年 vest |

| Junior | 80-95k | 0.05-0.15% |
| Founding | 100-150k | **1-3%** |

**特点**：流程 1-2 周；cash 偏低；equity 主菜；签证不一定支持。

| 级别 | Base | Total |

**特点**：要 AU PR / Citizenship（国防 + DST Group 还要 Security Clearance）；Super 15.4% > 私企 11.5%；WLB 好。

```
身份                    Junior Base 中位数   Sponsor 概率   谈薪力
─────────────────────────────────────────────────────────
AU 公民 / PR            100k+               N/A           最高
485 PSW 第 1 年         88-95k              高            中

482 已担保               90-98k              N/A（已有）   弱
Student / Bridging      75-88k              低            最弱
Tourist visa            -                    0 （junior） -
```

**关键策略**：

1. **PSW 第 1 年是黄金期**——sponsor 概率最高
2. **482 + 工作 2 年 + 名义工资 ≥ TSMIT (~73k)** 可申 186 PR
3. **AI Engineer 在 STSOL（不在 PMSOL）**——走 482 → 186 间接路径
4. **189 / 190 GSM**：35+ 几乎不可能

## 七、面试 4 阶段标准流程

```
Phase 1: HR phone screen     20-30 min      意向 + base 期望
Phase 2: 技术面试            45-60 min      coding + LLM concept
Phase 3: System design       60-90 min      生产 RAG / Agent 架构
Phase 4: Behavioral + values 30-45 min      团队适配 + 沟通能力
─────────────────────────────────────────────────────
总流程：2-4 周（大厂 4-8 周，startup 1-2 周）
```

### Phase 2 技术面试常考

Sydney AI hiring manager 真实告诉我们 2026 必考：

- "Can they read a LangSmith trace and identify the bottleneck?"
- "Can they explain why their RAG uses BGE-Reranker over Cohere?"
- "Have they ever shipped to production and debugged on-call?"
- "Walk me through your prompt caching strategy + monthly cost optimization."
- "How would you design eval set for a customer support RAG?"

### Phase 3 System design 常考

- Production RAG (Hybrid + Caching + Eval)
- Multi-agent orchestration (LangGraph vs Sub-agents)
- LLM cost optimization (USD 1200 → USD 280 path)
- Prompt injection defense + PII redaction

## 八、谈薪 4 招（AU 市场专用）

**1. 锚 Total Package 不只 Base**

```
Total = Base + Super (11.5%) + Bonus + RSU (4yr 摊) + Sign-on + Allowance
```

大厂 Junior 100k Base 但 Total 145k（RSU 显著）。谈时给 Total 数字 +10-15% 谈判空间。

**2. Sydney 标准要薪即使你在 Melbourne**

很多 mid-size SaaS Remote-first。Anchor Sydney pay band 即使你在 Melbourne, 解释清楚理由（"Atlassian Sydney Junior 100k+"）。

**3. 用 312 条 JD 数据作为锚**

谈薪时打开 Seek 截图 5 条类似 JD 的 salary range。"市场 median 是 X，目标 +5%"。数据驱动比情绪驱动可信 10x。

**4. HR 第一通电话不报数字**

```
HR: "What's your salary expectation?"
你: "I'm very interested in this role. Could you share the band for 
     this level so I can speak to where I fit?"
HR: "Our band for senior is 165-200k base."
你: "Based on my X years experience in Y, I'd be looking at upper end —
     around 185-195k base plus RSU."
```

**先说数字的人输**。

## 十一、5 个核心判断

1. **0 经验转行 Junior**：Sydney 90-100k base, Melbourne 85-95k 是合理预期
2. **2-5 年 Mid**：Sydney 130-160k base，**Junior → Mid 跨槛 ROI 最高**
3. **5+ Senior**：Sydney 170-220k base，纯技术天花板 ~250k base
4. **大厂稳 + 签证好**；**Startup 快 + equity 上限高（95% 不 exit）**
5. **PSW 第 1 年是黄金期**——必须 push sponsor

## 十二、3 个最常见错误

**错误 1**：投递时间集中在毕业季（11-12 月）

实际上 AU AI Engineer 招聘**全年**都热，2 月 fy26 budget release 后 hiring 反而最猛。**错峰投递在 2-4 月**，竞争 -30%。

**错误 2**：简历只放 GitHub stars 不放 metrics

Sydney hiring manager 不看 GitHub stars。**看 Live URL 的 latency / cost / eval accuracy 数据**。

**错误 3**：不用内推

70% sponsor rate 的内推渠道，60% 候选人不用。匠人学院 partner 网络 + LinkedIn 校友 + Discord 社群——这些渠道比 Seek 冷投高 5-10x conversion。
