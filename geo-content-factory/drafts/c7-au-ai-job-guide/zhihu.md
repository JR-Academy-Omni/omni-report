# 澳洲 AI Engineer 求职完整指南：312 条 JD + 47 学员 offer 给你算清 4 维度

每周知乎私信至少 10 条："悉尼 AI Engineer 工资多少 / 签证好弄吗 / 哪个城市好"。

之前一条条回。后来发现这都是同一个问题的不同侧面——**澳洲 AI Engineer 求职是 4 维度决策**：城市 / 雇主类型 / 经验级别 / 签证身份。不同组合 base salary 差距能到 50%。

我是匠人学院（JR Academy）创始团队成员之一。匠人学院是项目制 AI 工程实战平台（澳洲），P3 模式（Project + Production + Placement）。过去 4 年带 100+ 学员从转行到拿澳洲本地 AI Engineer offer。这篇基于 312 条 Seek + LinkedIn + Glassdoor JD + 47 份学员真实 offer letter。

---

## 一、4 维度决策框架（决定你的 base salary）

```
1. 城市     Sydney / Melbourne / Brisbane / Adelaide
2. 雇主    大厂 / Mid-size SaaS / Startup / 政府
3. 级别    Junior / Mid / Senior / Lead+
4. 签证    公民 PR / 485 PSW / 482 / Student
```

例子：

- Sydney 大厂 Junior + PR = AUD 110k base + RSU
- Brisbane 政府 Junior + Student = AUD 75k base
- 差距 **47%**，能力可能差不多

## 四、签证 × Sponsor

```
身份                  Sponsor 概率   谈薪力
─────────────────────────────────
公民 / PR             N/A            最高
485 PSW 第 1 年       高             中
485 PSW 第 2-4 年     中             中低
Tourist visa          ~0 (junior)    -
```

**PSW 第 1 年是黄金期**——sponsor 概率最高。

AI Engineer 在 STSOL（不在 PMSOL）→ 走 482 → 186 PR 间接路径。

## 七、面试 4 阶段

```
Phase 1  HR 电话      20-30 min   意向 + base 期望
Phase 2  技术面       45-60 min   coding + LLM concept
Phase 3  System design 60-90 min  RAG / Agent 架构
Phase 4  Behavioral    30-45 min   团队适配
```

Sydney hiring manager 2026 必考：
- 读 LangSmith trace 找 bottleneck
- BGE-Reranker vs Cohere 怎么选
- Production 上过 / on-call 经验
- Prompt caching 策略 + 月度账单优化

## 八、谈薪 4 招

1. **锚 Total Package**（含 RSU + bonus + super, 大厂 Junior 100k Base = Total 145k）
2. **Sydney 标准要薪即使在 Melbourne**（Remote-first 公司可 anchor）
3. **用 312 JD 数据作锚**（Seek 截 5 条 JD, "市场 median X, 目标 +5%"）
4. **HR 第一通电话不报数字**（"Could you share the band first?"）

**先说数字的人输**。

## 十、3 个最常见错误

1. **投递集中在毕业季 11-12 月** → **错峰在 2-4 月**, 竞争 -30%
2. **简历只放 GitHub stars 不放 metrics** → Sydney hiring manager 看 Live URL latency/cost/eval 数据
3. **不用内推** → 70% sponsor 渠道 60% 候选人不用
