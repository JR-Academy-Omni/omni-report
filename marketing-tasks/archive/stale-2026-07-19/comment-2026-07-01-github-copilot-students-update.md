## 草稿

For students switching to the updated Copilot plan: the model picker now includes Claude Sonnet 5 (added June 30) alongside GPT-5 and Gemini 2.5 Pro. Sonnet 5 is worth trying for large-codebase explanation tasks — the 1M-token context window handles significantly larger repos without truncation, compared to Sonnet 4.6's 200k which would silently drop earlier files. For CLI-heavy workflows, the terminal Claude Code integration has separate quota rules from in-IDE Copilot, so check both if you're hitting limits.
