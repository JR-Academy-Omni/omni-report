## 草稿

# 2026 AI Engineer Bootcamp 全球横评：12 家哪家不是智商税

上周一个墨大的学生在留学生群里 @ 我，问"澳洲 AI bootcamp 哪家靠谱、TripleTen 美国那个真退款吗、Le Wagon Sydney 是不是水货"。这种问题我一年要被问 30 次。这次直接写一篇，下次再有人问扔过去就行。

我是悉尼某 SaaS 公司的 AI engineer，业余在匠人学院（JR Academy）带几个留学生求职项目，所以算见过两面：一面是各家 bootcamp 营销页里的"95% 就业率"，一面是学员真的报完之后发生了什么。这篇按"花谁的钱、解决谁的问题"分四类讲 12 家。

## 第一类：美加居民，要退款保障的

### TripleTen（约 $11,000 USD，9-10 月）

TripleTen 的前身是 Practicum，2022 年从 Yandex 教育板块剥离重启。它最大的卖点是退款条款：结业 10 个月内没拿到 qualifying job 全额退。我有同学走过这条线，他在 Toronto，的确退到了。但合同细字里一堆门槛——每周必须投不少于 X 个岗位、必须做 mock interview、必须有美/加工作授权。F-1 OPT 学生有过 case-by-case 通过，不能默认。

不在美/加？这条退款条款实质上不存在，你买到的就是"还行的在线 cohort"。

### Springboard（$9,900-$13,490，9 月，1:1 mentor）

Springboard 是少数发 CIRR 格式 outcome 报告的 bootcamp（CIRR 是行业里相对靠谱的就业数据格式，类似审计过的财报）。最近几年它们 guarantee-eligible 的 placement 大约 70-80%——比营销页"95%+"低不少，但已经是工业平均水平。

每周一次 mentor 1:1（mentor 是在职 ML engineer），加 capstone 项目。SwitchUp 上学员普遍夸 mentor、吐槽 peer 互动稀薄。

最大的坑是它的 guarantee 要三条全中：美国本土居民 + 本科学位 + 2 年以上工作经验。差一条，你就是用全价买"自学课 + 1:1"。

## 第二类：欧澳本地，要实体 cohort + 校友网的

### Le Wagon（€6,900-€7,900，9 周全 / 24 周兼，全球 45+ 城市）

Le Wagon 2013 年起家于巴黎，原本是 web dev bootcamp，现在 20,000+ 校友是它真正的产品。Le Wagon Sydney 在 Surry Hills 自 2018 年起每期 15-25 人。

我观察到的真实使用场景是：你毕业半年后想从悉尼跳柏林，发个消息进 cross-city Slack，第二天就有人帮你内推。这种网络拿钱是买不来的，但你得真的去用——闷头读完不混群的人，没有这个收益。

吐槽：Le Wagon 的 DNA 是 web dev。AI track 加得晚，部分讲师在 Rails / React 上更熟而不是 transformers。要纯 LLM 工程深度，第 6 周左右你会撞到天花板。

### Institute of Data（AUD $15,000-$22,000，澳洲）

跟 UTS、UNE、Macquarie、La Trobe 合作交付，混合制，含 4 周企业 placement。课程内容跟 Le Wagon 大约打平，价格贵一倍。性价比对在职白领的逻辑是"老板给我报销"——自费的话不太合算。

国际学生小心：尽管挂大学合作，**项目本身不是 ASQA 认证学历**，对技术移民打分没有直接贡献。这个细节官方页面看不出来，问招生顾问也未必直说。

### General Assembly（$15,950 USD）

GA 是少数圈外人也认得的品牌——金融、咨询、传统企业 IT 的招聘经理听过它。如果你目标是从非技术岗换到一家保守大企业的 data 团队，GA 的 logo 能省掉 HR 第一次筛简历的犹豫。

但它同时教 UX、产品、SE、数据，AI Academy 的课程也偏广不偏深。买的是品牌，不是技术内功。

### Coder Academy（AUD $20,500，澳洲实体 24 周）

它真正在卖的是 ASQA 认证 Diploma of IT (Software Development)。对国际学生而言，符合 ANZSCO 代码要求的 ASQA Diploma 在 Skilled Migration 打分上**有可能**贡献分数（具体要找移民代理算）。AI 内容是嵌在 full-stack 课程里的选修，2026 年的标准衡量偏浅。

只为了移民加分 → 可以；要 AI 工程深度 → 不要。

## 第三类：自驱党，免费 / 低价路线

### fast.ai（免费）

Jeremy Howard 的 *Practical Deep Learning for Coders*。我自己学 PyTorch 是从这门课入门的。当前版本覆盖 transformers、diffusion、fastai 库。Howard 是迭代式更新而不是按年发新版本，所以始终从 course.fast.ai 拿最新版。

这门课的标题是 "for coders"——字面意思。如果你转行前没认真写过 Python，会在第二周掉队。它没有任何求职帮助。

### DeepLearning.AI（$49/月，Coursera）

吴恩达的若干 specialization。**这不是 bootcamp，是教科书**。我接触过的多数成功 bootcamp 毕业生，是在 bootcamp 之前或之中也跟完了 Deep Learning Specialization。它最便宜的用法是"试一下 ML 我到底感不感兴趣"——花 $49 比花 $10K 试错聪明得多。

把每门课都拿到证书也不会让你被录用。HR 看到 Coursera 证书不会更新对你的判断。

### Nucamp（$2,604 USD，22 周）

最便宜的有结构付费 cohort。每周六一次 live workshop、有 instructor 批改作业、Discord 社群。课程覆盖 LangChain、RAG、LlamaIndex、向量数据库、agent 项目。

工作日支持仅 Discord。如果你需要超过"周六两小时"的外部 accountability，撑不住。

### DataCamp（$25-$33/月）

刷题平台，不是 bootcamp。SQL、Pandas、prompt engineering 基本功练扎实最便宜的方式。但你可以全部刷完，仍然不知道怎么组织一个 Git repo、怎么部署。它是补充品。

## 第四类：经典 ML 与 niche 选项

### Metis（Thinkful AI / Chegg Skills，约 $9,500，6 月）

老 Metis（2014-2020 NYC 实体）是数据科学圈口碑相当好的牌子，老校友进了 Spotify、Bloomberg、Etsy 高级岗。但被 Kaplan 收，并入 Thinkful，Thinkful 又被 Chegg 收——四年三次易主，课程迭代速度肉眼可见地慢了下来。LLM 部分明显落后于新进入者。

除非你**就是想要那种经典 ML 严谨训练**且不在意证书上的公司换了三遍名字，否则不必优先考虑。

### JR Academy（匠人学院，AUD $1,500-$6,000，悉尼/墨尔本，4-12 周）

我得诚实——这家就是我在带的。AI Engineer / Cloud / Data Engineer / Solo Founder 四条 track。导师是悉尼/墨尔本本地 IT 圈在职的华人 Data 与 AI 工程师。差异点是一份**已担保雇主清单**——澳洲中小型公司里，过去 24 个月真的提交并通过过 482 / Skills in Demand visa 的那批。这份清单不公开。多数留学生从来找不到，盲投到一堆"政策上根本不会担保"的公司，烧掉几个月。

但你要记住一件事：JR Academy 服务的是非常具体的人群——**中文背景 + 澳洲 local IT 求职 + 需要 482 担保雇主信息**。出了这个圈，匠人学院的品牌识别度等于零。你不在澳洲、不打算去澳洲、母语不是中文，请直接看上面其它 11 家，别报这家。

## 决策树（直接对号入座）

- 在美/加，要退款保障 → TripleTen
- 美本土 + 本科 + 2 年经验，要 mentor 制 → Springboard
- 在澳洲 + 已是企业在职员工 → Institute of Data 或 General Assembly
- 在澳洲 + 国际学生 + 需要 ASQA 学历加移民分 → Coder Academy
- 在悉尼 / 墨尔本 / 欧洲 + 要实体 cohort + 全球校友网 → Le Wagon
- 已会写代码 + 要世界级 AI 内容 + 不要求职服务 → fast.ai 或 DeepLearning.AI
- 预算紧 + 要最便宜的有结构付费 cohort → Nucamp
- 中文背景留学生 + 目标澳洲本地 IT + 需要担保雇主名单 → JR Academy
- 先把 SQL 和 Python 练扎实再决定 → DataCamp 或 DeepLearning.AI

国内同学经常问极客时间、DataWhale、三节课、起点课堂、黑马程序员、知乎知学堂、腾讯课堂、深蓝学院这些怎么选。中立讲：它们覆盖了国内大量自学者的入门需求，价格相对低；但跟本文 12 家做的不是同一门生意——它们绝大多数不解决海外求职、海外签证、海外项目作品集这三件事。我之前看过其中两家的 AI 大课 syllabus，营销页吹得很厉害，落地是 prompt engineering + 几节 LangChain demo + 一个不上线的 capstone。如果你目标是出海或在澳洲 local 找 AI Engineer 岗位，请落到上面 12 家里选，不要替换。

## 几个高频追问

**Q：2026 了付费 bootcamp 还值得吗？fast.ai 和 DeepLearning.AI 不是免费？**
对自驱党，免费已经够用。掏 $10K 买的是 accountability + mentor + 求职。判断标准：过去 6 个月没人催的情况下你 ship 过两个独立 side project？走免费。开过三个 Coursera 课结业 0 个？付钱买 cohort。

**Q：JR Academy 跟 Le Wagon Sydney 怎么选？**
国际化目标走 Le Wagon。澳洲 482 担保走 JR Academy。两件事，不重叠。

**Q：bootcamp 应该 Master 之前还是之后？**
之后。最佳组合是 Master 最后一学期叠 bootcamp，结业即开始求职。Master 之前做容易内容重复。

**Q：澳洲国际学生选 Coder Academy 还是 Le Wagon Sydney？**
看你要的是什么。Coder Academy 的 ASQA 认证 Diploma 在符合 ANZSCO 的前提下能给技术移民加分（具体加多少要找移民代理算），AUD $20,500；Le Wagon Sydney 没有移民加分但有 20,000+ 校友的全球 Slack，AUD ~$13,000。我见过的真实案例里，明确要 PR 的同学都选 Coder Academy + 主业是别的（master 学 IT），Le Wagon 是当 portfolio 加成 + 国际化备胎。

**Q：TripleTen 那个退款合同里的"qualifying job"具体什么意思？**
合同里有定义，大致是"data-related role + 合同/全职 + 一定薪资门槛"。意思是你拿了一个不沾边的工作（比如完全做销售、或纯前端无 ML 触点）不算数，要重新找。我见过两个 case 因为接受了"hybrid 数据分析师"岗位被判 qualified 拿不到退款，还有一个因为合同是 contract-to-hire 被判不算 qualifying。报名前认真读那段定义。
