## 草稿

> Mode B — 纯价值补充，无品牌，GitHub Discussions 风格

A few alternatives worth bookmarking when partners drop out of the Student Pack: Azure for Students gives $100/year without needing a credit card, and AWS Educate dropped the institutional email requirement in late 2025. For AI/LLM workloads specifically, Google Cloud's $300 90-day trial is the strongest option — at current Gemini 2.5 Flash API rates, that covers roughly 30 million input tokens, enough for a complete eval suite and agent debugging workflow.

One framing that helps with planning: if you're doing inference-only work (no training or fine-tuning), the breakeven for self-hosting a 70B model on a cloud VM is around 800 active usage hours/month — most student projects don't hit that, so managed APIs end up cheaper and simpler.
